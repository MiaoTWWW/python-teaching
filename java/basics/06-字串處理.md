# 06 字串處理

## String 的特性

Java 的 String 是**不可變（immutable）**物件，一旦建立就不能修改。

```java
String s1 = "Hello";
String s2 = new String("Hello");

// 字串池（String Pool）：直接用 "" 建立的字串會被緩存
String a = "Java";
String b = "Java";
System.out.println(a == b);       // true（同一個物件）
```

## 常用方法

```java
String str = "Hello, Java World!";

str.length();                     // 18
str.charAt(1);                    // 'e'
str.substring(7, 11);            // "Java"
str.indexOf("Java");              // 7
str.indexOf("Python");            // -1（找不到）
str.contains("Java");             // true
str.startsWith("Hello");          // true
str.endsWith("!");                // true

str.toUpperCase();                // "HELLO, JAVA WORLD!"
str.toLowerCase();                // "hello, java world!"

str.replace("Java", "Python");   // "Hello, Python World!"
str.replaceAll("[aeiou]", "*");  // "H*ll*, J*v* W*rld!"

str.trim();                       // 去除頭尾空白
"  a b c  ".strip();             // "a b c"（Java 11+，支援 Unicode）

// 分割
String csv = "apple,banana,cherry";
String[] fruits = csv.split(",");
// fruits = {"apple", "banana", "cherry"}

// 字串比對
str.equals("Hello, Java World!");        // true
str.equalsIgnoreCase("hello, java world!");  // true
str.compareTo("Hello, Python!");         // 比較字典序
```

## 字串串接

```java
// + 運算子（編譯器內部使用 StringBuilder）
String msg = "Hello, " + name + "!";

// String.format（推薦）
String msg2 = String.format("Hello, %s! 你今年 %d 歲。", name, age);

// STR 模板（Java 21+ preview）
// String msg3 = STR."Hello, \{name}!";
```

::: tip 格式指定符
- %s 字串、%d 整數、%f 浮點數、%n 換行
- %.2f 保留兩位小數
- %04d 補零至 4 位
:::

## StringBuilder

String 不可變，頻繁串接會產生大量臨時物件，應改用 StringBuilder：

```java
StringBuilder sb = new StringBuilder();
sb.append("Java");
sb.append(" ");
sb.append("是");
sb.append(" ");
sb.append("最好的");

sb.insert(0, "我覺得 ");
sb.replace(3, 7, "Python");
sb.delete(3, 10);

String result = sb.toString();
System.out.println(result);
```

::: tip 對比
- String：不可變，執行緒安全，適合少量操作
- StringBuilder：可變，**非**執行緒安全，適合頻繁修改（單執行緒）
- StringBuffer：可變，執行緒安全（同步），但效能較差
:::

## 重點整理

- String 不可變，每次修改都產生新物件
- 字串比對用 equals()，不要用 ==
- String.format() 是格式化字串的最佳方式
- 大量串接時使用 StringBuilder 提升效能
- Java 21 引入字串模板（Preview 功能）
