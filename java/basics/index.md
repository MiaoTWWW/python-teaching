# 基礎篇

> 從完全不會 Java 到寫得出能動的程式。本章所有程式碼皆可直接複製貼上編譯執行。

## 適用對象

- 已有其他程式語言基礎，想快速上手 Java 語法
- 完全初學者 — 每一章都從零開始

## 章節地圖

| 章節 | 主題 | 你將學會 |
|------|------|---------|
| **01** | 環境安裝與設定 | JDK 安裝、環境變數、第一支 Hello World、編譯與執行原理、IDE 設定 |
| **02** | 變數與資料型別 | 八種基本型別、常數、型別轉換、var、Math 類別、**Wrapper Classes 與 autoboxing** |
| **03** | 流程控制 | if-else / switch-case 條件判斷、for / while / do-while 迴圈、break / continue、**Scanner 輸入**、**printf 格式化輸出** |
| **04** | 函式 | main 方法、參數傳遞、回傳值、方法多載、遞迴基礎 |
| **05** | 資料結構 | 一維 / 多維陣列宣告與操作、**Arrays 工具類別**（sort / binarySearch / fill / copyOf） |
| **06** | 字串處理 | String 不可變特性、常用方法（substring / replace / split / join）、StringBuilder / StringBuffer 可變字串 |
| **07** | 運算子 | 算術、關係、邏輯、位元運算、指派、條件（三元）運算子、運算子優先級 |

## 學習路徑

```
01 環境安裝
    ↓
02 變數型別 ← 先懂資料怎麼存
    ↓
07 運算子 ← 再懂資料怎麼算
    ↓
03 流程控制 ← 然後讓程式有判斷
    ↓
04 函式 ← 把程式模組化
    ↓
05 資料結構 + 06 字串處理 ← 處理批量資料與文字
```

::: tip
章節編號只是建議順序。建議至少依 01 → 02 → 03 → 04 的順序閱讀，05～07 可按需穿插。
:::

## 學完之後你能做到

- 看懂任何 Java 基礎程式的語法結構
- 獨立寫出判斷、迴圈、陣列操作、字串處理的完整程式
- 使用 Scanner 接收使用者輸入，用 printf 格式輸出
- 了解基本型別與包裝類別的差異與轉換
- 為進入**物件導向篇**做好準備
