# 01 環境安裝與設定

## 下載 JDK

Java Development Kit（JDK）是開發 Java 程式的必要工具包，內含編譯器（javac）與執行環境（JRE）。

目前主流版本為 **JDK 21（LTS）**，可從以下來源下載：

- Oracle JDK：https://www.oracle.com/java/technologies/downloads/
- OpenJDK（免費開源）：https://jdk.java.net/21/
- Amazon Corretto：https://aws.amazon.com/corretto/

::: tip 建議
初學者建議使用 **Oracle JDK 21 LTS**，安裝最簡單。
:::

## 安裝步驟（Windows）

1. 執行安裝程式，一路按「下一步」
2. 記下安裝路徑，預設為 C:\Program Files\Java\jdk-21
3. 設定環境變數：

`powershell
# 系統變數 → 新增
JAVA_HOME = C:\Program Files\Java\jdk-21

# 系統變數 → Path → 新增
%JAVA_HOME%\bin
```

## 驗證安裝

開啟命令提示字元，輸入：

`ash
java -version
javac -version
```

正確輸出範例：

```
java version "21.0.2" 2024-01-16 LTS
javac 21.0.2
```

## 第一個 Java 程式

建立 Hello.java：

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

編譯與執行：

`ash
javac Hello.java   # 產生 Hello.class
java Hello         # 執行，輸出：Hello, World!
```

::: tip
Java 檔案名稱必須與類別名稱**完全一致**，否則編譯失敗。
:::

## 開發工具

- **IntelliJ IDEA Community Edition**（推薦）：功能完整，對 Java 支援最佳
- **VS Code + Extension Pack for Java**：輕量，適合快速編輯
- **Eclipse**：老牌 IDE，仍廣泛使用

### IntelliJ IDEA 快速入門

1. 新增專案 → 選擇 JDK 21
2. 建立類別（New → Java Class）
3. 撰寫 main 方法，按 ▶ 執行

## 重點整理

- JDK = 開發工具，JRE = 執行環境，JVM = 虛擬機器
- javac 編譯原始碼成 .class 位元碼
- java 啟動 JVM 執行位元碼
- 一次編寫，到處執行（Write Once, Run Anywhere）
