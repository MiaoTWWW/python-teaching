# 07 運算子（Operators）

## 算術運算子

| 運算子 | 說明 | 範例 | 結果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 減法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法（整數除法截斷小數） | `7 / 3` | `2` |
| `%` | 取餘數（模數） | `7 % 3` | `1` |

```java
int a = 10, b = 3;
System.out.println(a + b);   // 13
System.out.println(a - b);   // 7
System.out.println(a * b);   // 30
System.out.println(a / b);   // 3（整數除法）
System.out.println(a % b);   // 1

// 浮點數除法
double c = 10.0, d = 3.0;
System.out.println(c / d);   // 3.333333...
```

::: warning
整數除以整數結果還是整數（小數直接截斷）。要得到浮點數結果，至少其中一個運算元要為浮點數。
:::

## 複合指定運算子

| 運算子 | 範例 | 等同於 |
|--------|------|--------|
| `+=` | `x += 5` | `x = x + 5` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 2` | `x = x * 2` |
| `/=` | `x /= 4` | `x = x / 4` |
| `%=` | `x %= 3` | `x = x % 3` |

## 遞增與遞減運算子

| 運算子 | 說明 | 範例 |
|--------|------|------|
| `++x` | 前綴：先加 1 再使用 | `int y = ++x;` |
| `x++` | 後綴：先使用再加 1 | `int y = x++;` |
| `--x` | 前綴：先減 1 再使用 | `int y = --x;` |
| `x--` | 後綴：先使用再減 1 | `int y = x--;` |

```java
int x = 5;
System.out.println(++x);  // 6（先加，再輸出）
System.out.println(x);    // 6

int y = 5;
System.out.println(y++);  // 5（先輸出，再加）
System.out.println(y);    // 6
```

## 關係運算子（比較）

| 運算子 | 說明 | 範例 |
|--------|------|------|
| `==` | 相等 | `a == b` |
| `!=` | 不相等 | `a != b` |
| `>` | 大於 | `a > b` |
| `<` | 小於 | `a < b` |
| `>=` | 大於等於 | `a >= b` |
| `<=` | 小於等於 | `a <= b` |

```java
int age = 18;
System.out.println(age >= 18);  // true
System.out.println(age == 20);  // false
```

::: warning
`==` 比較基本型別時比較「值」；比較參考型別時比較「記憶體位址」。字串比對應使用 `equals()`。
:::

## 邏輯運算子

| 運算子 | 說明 | 範例 |
|--------|------|------|
| `&&` | 短路 AND（左側 false 就不評估右側） | `a > 0 && b > 0` |
| `||` | 短路 OR（左側 true 就不評估右側） | `a > 0 || b > 0` |
| `!` | NOT（反相） | `!isDone` |

```java
int score = 85;
boolean hasPassed = score >= 60;       // true
boolean isExcellent = score >= 90;     // false
boolean canAdvance = hasPassed && !isExcellent;  // true

// 短路特性
String s = null;
if (s != null && s.length() > 0) {   // 安全：先檢查 null
    System.out.println("非空字串");
}
```

### 非短路版：`&` `|` `^`

| 運算子 | 說明 |
|--------|------|
| `&` | AND（兩側都評估） |
| `|` | OR（兩側都評估） |
| `^` | XOR（互斥或：兩側不同才為 true） |

布林邏輯中，`&` 與 `|` 和 `&&` `||` 的差別在於**是否短路**：

```java
int x = 5;
if (x > 10 & x++ > 0) { }   // x 變成 6（兩側都執行）
System.out.println(x);       // 6

int y = 5;
if (y > 10 && y++ > 0) { }  // y 仍是 5（短路，右側跳過）
System.out.println(y);       // 5
```

## 位元運算子

直接操作整數的二進位位元，常用於權限標記與效能優化：

| 運算子 | 說明 | 範例 |
|--------|------|------|
| `&` | 位元 AND | `5 & 3` → `1` |
| `|` | 位元 OR | `5 | 3` → `7` |
| `^` | 位元 XOR | `5 ^ 3` → `6` |
| `~` | 位元 NOT（補數） | `~5` → `-6` |
| `<<` | 左移（乘以 2 的冪） | `5 << 1` → `10` |
| `>>` | 右移（除以 2 的冪） | `5 >> 1` → `2` |
| `>>>` | 無號右移 | `-5 >>> 1` → 大正數 |

```java
int a = 5;   // 0101（二進位）
int b = 3;   // 0011

System.out.println(a & b);   // 0001 → 1
System.out.println(a | b);   // 0111 → 7
System.out.println(a ^ b);   // 0110 → 6
System.out.println(~a);      // ...11111010 → -6

// 位移運算
System.out.println(5 << 2);   // 20（5 * 4）
System.out.println(16 >> 2);  // 4（16 / 4）
```

::: tip
`x << n` 等同於 `x * 2ⁿ`，`x >> n` 等同於 `x / 2ⁿ`（整數除法）。
:::

## 三元運算子（Ternary）

唯一的三元運算子，是 if-else 的簡化形式：

```java
// 語法：條件 ? 成立時的值 : 不成立時的值
int age = 20;
String status = (age >= 18) ? "成年" : "未成年";
System.out.println(status);  // 成年

// 巢狀三元（不建議太長）
int score = 85;
String grade = score >= 90 ? "A" : (score >= 80 ? "B" : "C");
System.out.println(grade);  // B
```

## instanceof 運算子

檢查物件是否為某個型別或其子型別的實例：

```java
String s = "Hello";
System.out.println(s instanceof String);       // true
System.out.println(s instanceof Object);       // true（String 是 Object 的子類別）
System.out.println(s instanceof Comparable);   // true（String 實作了 Comparable）

// Java 16+ pattern matching
if (s instanceof String str) {
    System.out.println(str.length());  // 不需另外轉型
}
```

## 運算子優先順序（由高到低）

| 優先級 | 運算子 |
|--------|--------|
| 1 | `()` `[]` `.` |
| 2 | `++` `--` `+`(正) `-`(負) `~` `!` |
| 3 | `*` `/` `%` |
| 4 | `+` `-` |
| 5 | `<<` `>>` `>>>` |
| 6 | `<` `>` `<=` `>=` `instanceof` |
| 7 | `==` `!=` |
| 8 | `&` |
| 9 | `^` |
| 10 | `|` |
| 11 | `&&` |
| 12 | `||` |
| 13 | `? :`（三元） |
| 14 | `=` `+=` `-=` `*=` `/=` `%=` `&=` `|=` `^=` `<<=` `>>=` `>>>=` |

::: tip
不確定優先順序時**加括號**，讓程式碼更易讀。
:::

## 重點整理

- 整數除法會截斷小數，需注意型別轉換
- `&&` `||` 有短路特性，`&` `|` 無短路
- `==` 比較參考型別時是比位址，不是比內容
- 三元運算子 `? :` 簡化簡單 if-else
- `instanceof` 檢查物件型別
- 善用括號明確表達運算順序
