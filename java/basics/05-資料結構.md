# 05 資料結構

## 陣列（Array）

陣列是固定長度的容器，所有元素型別必須相同：

```java
// 宣告 + 初始化
int[] numbers = new int[5];           // {0, 0, 0, 0, 0}
int[] scores = {85, 92, 78, 90};      // 直接給值

// 存取與修改
scores[0] = 88;
System.out.println(scores[0]);        // 88
System.out.println(scores.length);    // 4

// 遍歷
for (int i = 0; i < scores.length; i++) {
    System.out.println(scores[i]);
}

// for-each
for (int score : scores) {
    System.out.println(score);
}
```

### 多維陣列

```java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(matrix[1][2]);  // 6 （第 2 列第 3 行）
```

## ArrayList（動態陣列）

ArrayList 是可動態增減長度的集合，需先 import java.util.ArrayList;：

```java
import java.util.ArrayList;

ArrayList<String> names = new ArrayList<>();

names.add("Alice");
names.add("Bob");
names.add("Charlie");

System.out.println(names.get(0));       // Alice
System.out.println(names.size());       // 3
names.remove(1);                         // 移除 Bob
names.set(0, "Alex");                    // 修改第 0 項

// 遍歷
for (String name : names) {
    System.out.println(name);
}
```

## HashMap（鍵值對）

```java
import java.util.HashMap;

HashMap<String, Integer> ages = new HashMap<>();

ages.put("Alice", 25);
ages.put("Bob", 30);
ages.put("Charlie", 28);

System.out.println(ages.get("Alice"));    // 25
System.out.println(ages.containsKey("Bob"));  // true
System.out.println(ages.size());          // 3

// 遍歷所有鍵值
for (String key : ages.keySet()) {
    System.out.println(key + " → " + ages.get(key));
}
```

## HashSet（不重複集合）

```java
import java.util.HashSet;

HashSet<String> uniqueNames = new HashSet<>();
uniqueNames.add("Alice");
uniqueNames.add("Bob");
uniqueNames.add("Alice");     // 重複，不會加入

System.out.println(uniqueNames.size());       // 2
System.out.println(uniqueNames.contains("Alice"));  // true
```

## 資料結構比較

| 結構 | 順序 | 可重複 | 鍵值對 | 適合場景 |
|------|------|--------|--------|----------|
| 陣列 | ✅ 固定順序 | ✅ | ❌ | 大小固定、高效索引 |
| ArrayList | ✅ 插入順序 | ✅ | ❌ | 動態增減的列表 |
| HashMap | ❌ 無順序 | 值可重複 | ✅ | 快速查找鍵→值 |
| HashSet | ❌ 無順序 | ❌ | ❌ | 去重、快速存在判斷 |

## Arrays 工具類別

`java.util.Arrays` 提供陣列相關的便利操作：

```java
import java.util.Arrays;

int[] arr = {5, 3, 1, 4, 2};

// 排序
Arrays.sort(arr);                     // [1, 2, 3, 4, 5]

// 轉為字串輸出
System.out.println(Arrays.toString(arr));  // [1, 2, 3, 4, 5]

// 複製陣列
int[] copy = Arrays.copyOf(arr, 3);   // [1, 2, 3]（只取前 3 個）
int[] range = Arrays.copyOfRange(arr, 1, 4);  // [2, 3, 4]（索引 1~3）

// 填滿
int[] filled = new int[5];
Arrays.fill(filled, 7);               // [7, 7, 7, 7, 7]

// 二元搜尋（陣列必須先排序）
int index = Arrays.binarySearch(arr, 3);  // 2（找到 3 的位置）

// 比對陣列是否相等
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
System.out.println(Arrays.equals(arr1, arr2));  // true
```

::: tip
`Arrays.toString()` 對除錯非常方便，可以一覽陣列內容，不用自己寫迴圈。
:::

## 重點整理

- 陣列長度固定，用 `length` 屬性取得長度
- ArrayList 需 `import java.util.ArrayList;`
- HashMap 以鍵值對儲存，鍵不可重複
- HashSet 自動過濾重複元素
- 使用泛型 `<型別>` 指定容器內的元素型別
- `Arrays.sort()`、`Arrays.toString()`、`Arrays.copyOf()` 是常用工具方法
