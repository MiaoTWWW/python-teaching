# 02 變數與資料型別

## 基本資料型別（Primitive Types）

Java 有 **8 種**基本型別，分別存放整數、浮點數、字元與布林值：

| 型別 | 大小 | 範圍 | 範例 |
|------|------|------|------|
| yte | 8 位元 | -128 ~ 127 | yte b = 100; |
| short | 16 位元 | -32,768 ~ 32,767 | short s = 1000; |
| int | 32 位元 | -2³¹ ~ 2³¹-1 | int i = 100; |
| long | 64 位元 | -2⁶³ ~ 2⁶³-1 | long l = 100L; |
| loat | 32 位元 | ±3.4E-38 ~ ±3.4E+38 | loat f = 3.14f; |
| double | 64 位元 | ±1.7E-308 ~ ±1.7E+308 | double d = 3.14; |
| char | 16 位元 | 0 ~ 65,535（Unicode） | char c = 'A'; |
| oolean | 1 位元（理論） | true / false | oolean flag = true; |

## 變數宣告

```java
int age = 25;
double price = 99.9;
char grade = 'A';
boolean isPass = true;
```

::: warning 注意
- long 結尾加 L，loat 結尾加 
- char 使用**單引號**，String 使用**雙引號**
:::

## 型別轉換（Casting）

### 隱式轉換（自動擴展）
```java
int i = 100;
long l = i;       // int → long 自動轉換
double d = l;     // long → double 自動轉換
```

### 明確轉換（強制縮小）
```java
double d = 99.9;
int i = (int) d;  // 99，小數被截斷
```

## 常數（final）

使用 inal 關鍵字宣告不可變的常數：

```java
final double PI = 3.14159;
final int MAX_SIZE = 100;
// PI = 3.0;  ← 編譯錯誤！不能修改
```

## var 關鍵字（Java 10+）

Java 10 引入區域變數型別推斷：

```java
var name = "Alice";          // 編譯器推斷為 String
var count = 42;              // 推斷為 int
var list = new ArrayList<String>();  // 推斷為 ArrayList<String>
```

::: tip
ar 只能用在**區域變數**，不能用在欄位、方法參數或回傳型別。
:::

## Math 常用類別

java.lang.Math 提供常見的數學運算（不需 import）：

```java
// 數值運算
Math.abs(-5);            // 5（絕對值）
Math.max(10, 20);        // 20（取大）
Math.min(10, 20);        // 10（取小）
Math.round(3.6);         // 4（四捨五入）
Math.floor(3.6);         // 3.0（無條件捨去）
Math.ceil(3.2);          // 4.0（無條件進位）

// 指數與對數
Math.pow(2, 10);         // 1024.0（2 的 10 次方）
Math.sqrt(16);           // 4.0（平方根）
Math.log(Math.E);        // 1.0（自然對數）
Math.log10(100);         // 2.0（常用對數）

// 三角函數（弧度）
Math.sin(Math.PI / 2);   // 1.0
Math.cos(0);             // 1.0
Math.toRadians(180);     // π（角度轉弧度）

// 隨機數
Math.random();           // 0.0 ~ 1.0 之間的隨機浮點數

// 範例：產生 1~100 的隨機整數
int rand = (int)(Math.random() * 100) + 1;
```
::: tip
Math.random() 回傳 double，範圍 [0.0, 1.0)。搭配乘法與 (int) 轉型可產生任意範圍的整數亂數。
:::

## Wrapper Classes（包裝類別）

Java 為每個基本型別都提供了一個對應的包裝類別，讓基本型別也能用在需要物件的場合（例如集合框架）。

| 基本型別 | 包裝類別 | 初始值 |
|---------|---------|--------|
| `byte` | `Byte` | `null` |
| `short` | `Short` | `null` |
| `int` | `Integer` | `null` |
| `long` | `Long` | `null` |
| `float` | `Float` | `null` |
| `double` | `Double` | `null` |
| `char` | `Character` | `null` |
| `boolean` | `Boolean` | `null` |

### Autoboxing 與 Unboxing

Java 5 以後，基本型別與包裝類別之間可以自動轉換：

```java
// Autoboxing：int → Integer
Integer num = 100;          // 編譯器自動轉換為 Integer.valueOf(100)

// Unboxing：Integer → int
int value = num;            // 編譯器自動轉換為 num.intValue()

// 在集合中的應用
List<Integer> scores = new ArrayList<>();
scores.add(95);             // autoboxing：int → Integer
int first = scores.get(0);  // unboxing：Integer → int
```

### 常用方法

```java
// 字串轉數字
int n1 = Integer.parseInt("123");       // 123
double d1 = Double.parseDouble("3.14"); // 3.14

// 數字轉字串
String s1 = Integer.toString(456);      // "456"
String s2 = Integer.toHexString(255);   // "ff"

// 常數
int maxInt = Integer.MAX_VALUE;         // 2147483647
int minInt = Integer.MIN_VALUE;         // -2147483648
```

::: tip
包裝類別是不可變的（immutable），一旦建立就不能改變裡面的值。`Integer`、`Long` 等也有快取機制，預設快取 -128 ~ 127 的整數物件。
:::

## 重點整理

- 基本型別儲存在**堆疊**（stack），直接存值
- 參考型別儲存在**堆積**（heap），變數存記憶體位址
- `final` 讓變數不可重新賦值
- `var` 簡化區域變數宣告，但不影響型別安全性
- 包裝類別讓基本型別可物件化，autoboxing/unboxing 提供自動轉換
