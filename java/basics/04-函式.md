# 04 函式（方法）

## 方法定義

Java 的方法必須定義在類別內部，基本語法：

```java
public static int add(int a, int b) {
    return a + b;
}
// 存取修飾詞  靜態   回傳型別  方法名(參數列表)
```

## 回傳值與 void

```java
public static int square(int x) {
    return x * x;       // 回傳 int
}

public static void greet(String name) {
    System.out.println("Hello, " + name);  // 不回傳值
}
```

## 參數傳遞：傳值（Call by Value）

Java **永遠是傳值**——基本型別傳值複本，參考型別傳位址複本：

```java
public static void changeValue(int x) {
    x = 100;  // 只修改區域變數，不影響外部
}

public static void main(String[] args) {
    int num = 10;
    changeValue(num);
    System.out.println(num);  // 10，沒變
}
```

但參考型別可修改物件內容：

```java
public static void changeName(String[] names) {
    names[0] = "Bob";
}

public static void main(String[] args) {
    String[] arr = {"Alice"};
    changeName(arr);
    System.out.println(arr[0]);  // Bob
}
```

## 方法重載（Overloading）

同名方法，參數數量或型別不同：

```java
public static int add(int a, int b) {
    return a + b;
}

public static double add(double a, double b) {
    return a + b;
}

public static int add(int a, int b, int c) {
    return a + b + c;
}
```

::: tip
編譯器根據**參數的數量與型別**決定呼叫哪個方法。
:::

## 變數作用域

```java
public static void example() {
    int x = 10;           // 方法層級

    if (x > 5) {
        int y = 20;       // 區塊層級，只能在 if 內使用
        System.out.println(x + y);
    }
    // System.out.println(y);  ← 編譯錯誤！y 已超出作用域
}
```

## 方法呼叫

- **靜態方法**：直接用 類別名.方法名() 呼叫
- **實例方法**：需建立物件後才可呼叫

```java
public class Calculator {
    // 靜態方法
    public static int add(int a, int b) {
        return a + b;
    }

    // 實例方法
    public int multiply(int a, int b) {
        return a * b;
    }

    public static void main(String[] args) {
        int sum = Calculator.add(3, 5);      // 靜態呼叫
        Calculator calc = new Calculator();
        int product = calc.multiply(4, 6);   // 實例呼叫
    }
}
```

## 重點整理

- 方法必須在類別內定義
- static 方法屬於類別，不需物件即可呼叫
- Java 只有**傳值呼叫**，沒有傳參考呼叫
- 方法重載讓同名方法有不同的參數版本
