# 02 集合框架

Java 的集合框架（Collections Framework）是一套統一的架構，用於儲存與操作群組資料。

## 集合層級架構

```
Iterable
  └─ Collection
        ├─ List（有序，可重複）
        │    ├─ ArrayList
        │    └─ LinkedList
        ├─ Set（無序，不可重複）
        │    ├─ HashSet
        │    ├─ LinkedHashSet
        │    └─ TreeSet
        └─ Queue（佇列）
             └─ PriorityQueue

Map（鍵值對，不屬於 Collection）
  ├─ HashMap
  ├─ LinkedHashMap
  └─ TreeMap
```

## List

```java
import java.util.*;

// ArrayList：底層陣列，隨機存取快，插入/刪除慢
List<String> arrayList = new ArrayList<>();
arrayList.add("Apple");
arrayList.add("Banana");
arrayList.add("Cherry");
System.out.println(arrayList.get(1));  // Banana

// LinkedList：底層雙向鏈結，插入/刪除快，隨機存取慢
List<String> linkedList = new LinkedList<>();
linkedList.add("Dog");
linkedList.add("Cat");
```

## Set（不重複集合）

```java
// HashSet：無順序，效能最佳
Set<String> hashSet = new HashSet<>();
hashSet.add("Apple");
hashSet.add("Banana");
hashSet.add("Apple");      // 不加入，set 長度仍為 2

// TreeSet：紅黑樹，自動排序（自然排序或 Comparator）
Set<String> treeSet = new TreeSet<>();
treeSet.add("Charlie");
treeSet.add("Alice");
treeSet.add("Bob");
System.out.println(treeSet);  // [Alice, Bob, Charlie]（字母排序）

// LinkedHashSet：保留插入順序
Set<String> linkedHashSet = new LinkedHashSet<>();
```

## Map（鍵值對）

```java
// HashMap：無順序，效能最佳
Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 95);
scores.put("Bob", 87);
scores.put("Charlie", 91);
System.out.println(scores.get("Bob"));        // 87
System.out.println(scores.containsKey("Alice"));  // true
System.out.println(scores.getOrDefault("David", 0));  // 0

// 遍歷 Map
for (Map.Entry<String, Integer> entry : scores.entrySet()) {
    System.out.println(entry.getKey() + " → " + entry.getValue());
}
```

## Collections 工具類

```java
List<Integer> list = new ArrayList<>(Arrays.asList(3, 1, 4, 1, 5));

Collections.sort(list);           // [1, 1, 3, 4, 5]
Collections.reverse(list);        // [5, 4, 3, 1, 1]
Collections.shuffle(list);        // 隨機打亂
Collections.max(list);            // 5
Collections.min(list);            // 1
Collections.frequency(list, 1);   // 2（1 出現次數）
```

## Comparable vs Comparator

```java
// Comparable：自然排序（實作 compareTo）
public class Student implements Comparable<Student> {
    String name;
    int score;

    @Override
    public int compareTo(Student other) {
        return Integer.compare(this.score, other.score);  // 依分數排序
    }
}

// Comparator：自訂排序
Comparator<Student> byName = new Comparator<>() {
    @Override
    public int compare(Student a, Student b) {
        return a.name.compareTo(b.name);
    }
};

// Lambda 版本
Comparator<Student> byNameDesc = (a, b) -> b.name.compareTo(a.name);

// 使用
List<Student> students = new ArrayList<>();
Collections.sort(students);                  // Comparable 排序
Collections.sort(students, byName);          // Comparator 排序
```

## 重點整理

- List 有序可重複、Set 無序不可重複、Map 鍵值對
- ArrayList 最常用、HashMap 最常用的 Map
- TreeSet / TreeMap 自動排序
- Collections 工具類提供排序、搜尋、同步包裝等靜態方法
- Comparable 定義自然排序，Comparator 定義自訂排序
