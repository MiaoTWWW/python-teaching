# 08 日期時間 API（java.time）

Java 8 引入全新的日期時間 API，解決了舊版 `java.util.Date` 與 `Calendar` 的各種設計問題。

## 舊版 API 的問題

```java
// ❌ 舊版問題
Date date = new Date(2024, 1, 1);   // 月份從 0 開始，不易讀
date.setYear(2025);                  // 可變物件，不安全
Calendar cal = Calendar.getInstance();
int month = cal.get(Calendar.MONTH); // 需要常數，繁瑣
```

## LocalDate（日期）

不帶時間的日期（年、月、日）：

```java
import java.time.LocalDate;

// 建立
LocalDate today = LocalDate.now();                    // 今天
LocalDate specific = LocalDate.of(2024, 12, 25);      // 2024-12-25
LocalDate parsed = LocalDate.parse("2024-12-25");     // 從字串解析

// 存取
specific.getYear();        // 2024
specific.getMonth();       // DECEMBER（Month 列舉）
specific.getMonthValue();  // 12（數字月份）
specific.getDayOfMonth();  // 25
specific.getDayOfWeek();   // WEDNESDAY（DayOfWeek 列舉）

// 運算（回傳新物件，原物件不變）
today.plusDays(7);         // 7 天後
today.minusMonths(1);      // 1 個月前
today.plusWeeks(2);        // 2 週後
today.plusYears(1);        // 1 年後

// 比較
today.isBefore(specific);  // true
today.isAfter(specific);   // false
today.isEqual(specific);   // false
```

## LocalTime（時間）

不帶日期的時間（時、分、秒、奈秒）：

```java
import java.time.LocalTime;

// 建立
LocalTime now = LocalTime.now();                      // 現在時間
LocalTime specific = LocalTime.of(14, 30, 0);         // 14:30:00
LocalTime parsed = LocalTime.parse("14:30:00");       // 從字串解析

// 存取
specific.getHour();        // 14
specific.getMinute();      // 30
specific.getSecond();      // 0

// 運算
now.plusHours(3);
now.minusMinutes(30);
```

## LocalDateTime（日期 + 時間）

同時包含日期與時間：

```java
import java.time.LocalDateTime;

// 建立
LocalDateTime now = LocalDateTime.now();
LocalDateTime specific = LocalDateTime.of(2024, 12, 25, 14, 30);
LocalDateTime parsed = LocalDateTime.parse("2024-12-25T14:30:00");

// 組合
LocalDate date = LocalDate.now();
LocalTime time = LocalTime.now();
LocalDateTime combined = LocalDateTime.of(date, time);

// 分解
LocalDate d = combined.toLocalDate();
LocalTime t = combined.toLocalTime();
```

## ZonedDateTime（含時區）

```java
import java.time.ZonedDateTime;
import java.time.ZoneId;

// 建立
ZonedDateTime now = ZonedDateTime.now();                      // 系統時區
ZonedDateTime tokyo = ZonedDateTime.now(ZoneId.of("Asia/Tokyo"));
ZonedDateTime parsed = ZonedDateTime.parse("2024-12-25T14:30:00+08:00[Asia/Taipei]");

// 轉換時區
ZonedDateTime taipei = ZonedDateTime.now(ZoneId.of("Asia/Taipei"));
ZonedDateTime ny = taipei.withZoneSameInstant(ZoneId.of("America/New_York"));
System.out.println(taipei + " 是紐約 " + ny);
```

## Duration 與 Period

時間間隔：

```java
import java.time.*;

// Duration：以秒/奈秒為單位（適合時間）
Duration d = Duration.between(
    LocalTime.of(9, 0), LocalTime.of(17, 30)
);
System.out.println(d.toHours());    // 8 小時
System.out.println(d.toMinutes());  // 510 分鐘

// Period：以年月日為單位（適合日期）
Period p = Period.between(
    LocalDate.of(2000, 1, 1), LocalDate.now()
);
System.out.println(p.getYears());   // 年齡（約數）
```

## DateTimeFormatter（格式化與解析）

```java
import java.time.format.DateTimeFormatter;

LocalDateTime now = LocalDateTime.now();

// 內建格式
String s1 = now.format(DateTimeFormatter.ISO_LOCAL_DATE_TIME);
// 2024-12-25T14:30:00

// 自訂格式
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy/MM/dd HH:mm:ss");
String s2 = now.format(formatter);
// 2024/12/25 14:30:00

// 解析字串
LocalDate date = LocalDate.parse("2024/12/25", DateTimeFormatter.ofPattern("yyyy/MM/dd"));
```

### 常用格式模式

| 符號 | 說明 | 範例 |
|------|------|------|
| `yyyy` | 四位數年份 | 2024 |
| `yy` | 兩位數年份 | 24 |
| `MM` | 兩位數月份 | 12 |
| `dd` | 兩位數日期 | 25 |
| `HH` | 24 小時制 | 14 |
| `hh` | 12 小時制 | 02 |
| `mm` | 分鐘 | 30 |
| `ss` | 秒 | 00 |
| `a` | 上午/下午 | 下午 |
| `EEE` | 星期縮寫 | 週三 |
| `EEEE` | 星期全名 | 星期三 |

## MonthDay 與 YearMonth

```java
// 只存月日（適合生日、節日）
MonthDay birthday = MonthDay.of(3, 15);
MonthDay christmas = MonthDay.parse("--12-25");

// 只存年月
YearMonth ym = YearMonth.of(2024, 12);
System.out.println(ym.lengthOfMonth());   // 31（該月有幾天）
System.out.println(ym.isLeapYear());      // true（閏年）
```

## 時間調整器（TemporalAdjusters）

```java
import java.time.temporal.TemporalAdjusters;

LocalDate today = LocalDate.now();

today.with(TemporalAdjusters.next(DayOfWeek.MONDAY));    // 下週一
today.with(TemporalAdjusters.firstDayOfMonth());          // 本月第一天
today.with(TemporalAdjusters.lastDayOfMonth());           // 本月最後一天
today.with(TemporalAdjusters.lastDayOfYear());            // 今年最後一天
today.with(TemporalAdjusters.firstInMonth(DayOfWeek.MONDAY)); // 本月第一個週一
```

## Instant（時間戳）

適合記錄事件發生的精確時間點：

```java
import java.time.Instant;

Instant now = Instant.now();                     // 當前 UTC 時間戳
System.out.println(now.getEpochSecond());        // 從 1970-01-01 起的秒數

// 與 Date 互相轉換
java.util.Date legacy = java.util.Date.from(now);
Instant back = legacy.toInstant();
```

## 重點整理

- 優先使用 `java.time` 套件，避免舊版 `Date` / `Calendar`
- `LocalDate`（日期）、`LocalTime`（時間）、`LocalDateTime`（兩者）
- 日期時間物件都是**不可變的**，每次運算回傳新物件
- `DateTimeFormatter.ofPattern()` 自訂格式
- `Duration` 計算時間間隔，`Period` 計算日期間隔
- `Instant` 適合記錄時間戳（UTC）
- `TemporalAdjusters` 提供常見日期調整（下週一、月底等）
