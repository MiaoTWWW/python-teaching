# 05 列舉（Enum）

## 為什麼需要列舉？

當變數只能取固定幾個值時，列舉比常數更安全、更易讀：

```java
// ❌ 使用常數：無法限制傳入值
public static final int SUNDAY = 0;
public static final int MONDAY = 1;
// ...
void setDay(int day) { }  // 傳入 100 也沒錯

// ✅ 使用列舉：編譯器檢查型別
enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
}

void setDay(Day day) { }  // 只能傳入 Day 型別的值
```

## 基本列舉

```java
public enum Color {
    RED, GREEN, BLUE
}

// 使用
Color c = Color.RED;
System.out.println(c);         // RED

// 遍歷所有列舉值
for (Color color : Color.values()) {
    System.out.println(color);
}
// RED  GREEN  BLUE
```

## 列舉常用方法

所有列舉都繼承 `java.lang.Enum`，提供以下方法：

```java
Color c = Color.RED;

c.name();              // "RED"（字串名稱）
c.ordinal();           // 0（宣告時的順序，從 0 開始）
c.toString();          // "RED"

Color.valueOf("GREEN");  // Color.GREEN（從字串轉列舉）
Color.values();          // [RED, GREEN, BLUE]（所有列舉值）

// 比對
c == Color.RED;       // true（列舉可用 == 比較）
c.equals(Color.RED);  // true
```

## 列舉搭配 switch

```java
Day day = Day.SATURDAY;

switch (day) {
    case MONDAY:
        System.out.println("上班日");
        break;
    case FRIDAY:
        System.out.println("週五小確幸");
        break;
    case SATURDAY:
    case SUNDAY:
        System.out.println("週末！");
        break;
    default:
        System.out.println("平日");
}

// 增強式 switch（Java 14+）
String type = switch (day) {
    case SATURDAY, SUNDAY -> "週末";
    case MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY -> "平日";
};
```

## 列舉可以有欄位與方法

列舉本質上是特殊的類別，可以定義建構子、欄位、方法：

```java
public enum Planet {
    MERCURY(3.303e23, 2.4397e6),
    VENUS(4.869e24, 6.0518e6),
    EARTH(5.976e24, 6.37814e6),
    MARS(6.421e23, 3.3972e6);

    private final double mass;       // 質量（kg）
    private final double radius;     // 半徑（m）

    // 建構子（必須是 private，列舉不能外部 new）
    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }

    // 方法
    public double getMass() { return mass; }
    public double getRadius() { return radius; }

    // 計算表面重力
    public double surfaceGravity() {
        final double G = 6.67300E-11;
        return G * mass / (radius * radius);
    }
}

// 使用
Planet earth = Planet.EARTH;
System.out.println(earth.getMass());      // 5.976E24
System.out.println(earth.surfaceGravity());
```

## 列舉可以實作介面

```java
public interface Describable {
    String getDescription();
}

public enum Status implements Describable {
    PENDING {
        @Override
        public String getDescription() {
            return "等待處理";
        }
    },
    PROCESSING {
        @Override
        public String getDescription() {
            return "處理中";
        }
    },
    COMPLETED {
        @Override
        public String getDescription() {
            return "已完成";
        }
    };
}

// 使用
Status s = Status.PENDING;
System.out.println(s.getDescription());  // 等待處理
```

## 實務範例：訂單狀態機

```java
public enum OrderStatus {
    NEW,
    PAID,
    SHIPPED,
    DELIVERED,
    CANCELLED;

    /** 是否允許轉換到目標狀態 */
    public boolean canTransitionTo(OrderStatus target) {
        return switch (this) {
            case NEW -> target == PAID || target == CANCELLED;
            case PAID -> target == SHIPPED || target == CANCELLED;
            case SHIPPED -> target == DELIVERED;
            case DELIVERED, CANCELLED -> false;
        };
    }
}

// 使用
OrderStatus current = OrderStatus.NEW;
if (current.canTransitionTo(OrderStatus.PAID)) {
    System.out.println("可以付款");
}
```

## 重點整理

- 列舉提供型別安全的常數集合
- 使用 `==` 比較列舉值（比 `equals()` 更簡潔）
- 列舉可以包含欄位、建構子、方法
- `values()` 取得所有列舉值陣列
- `valueOf(String)` 從字串轉列舉
- 列舉可以實作介面，不能繼承類別
- switch 搭配列舉非常適合實作狀態機
