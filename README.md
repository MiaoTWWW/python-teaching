# 📚 程式語言講義

> **📖 線上教材：https://MiaoTWWW.github.io/python-teaching/**

從零開始 · 邁向實戰 — 一份完整、易懂、結構清晰的程式語言學習教材，涵蓋 **Python**、**Java**、**C 語言**，適合初學者與想扎穩根基的開發者。

---

## 內容架構

### 🐍 Python

| 篇章 | 主題 |
|------|------|
| **基礎篇** | 環境安裝、變數與資料型別、流程控制、函式、資料結構、字串處理 |
| **進階篇** | 物件導向、模組與套件、錯誤處理、檔案 I/O、迭代器與生成器、閉包、裝飾器 |
| **實戰篇** | 網路爬蟲、資料庫操作、Web API 開發、資料科學入門、自動化腳本、執行緒與並行程式、ITS 認證準備 |
| **演算法篇** | 雜湊表、搜尋法、排序演算法、佇列、堆疊、樹、圖形、遞迴、動態規劃、進階模組 |

### ☕ Java

| 篇章 | 主題 |
|------|------|
| **基礎篇**（7 章） | 環境安裝、變數與資料型別、流程控制、函式、資料結構、字串處理、運算子 |
| **物件導向篇**（6 章） | 類別與物件、繼承與多型、介面與抽象類別、套件與存取修飾詞、列舉、巢狀類別 |
| **進階語法特色**（12 章） | 例外處理、集合框架、泛型、Lambda 與 Stream、多執行緒、JVM 記憶體管理、註解與反射、日期時間 API、單元測試、檔案 I/O、JDBC 資料庫操作、Spring Boot 後端藍圖 |

### 🔧 C 語言（完整基礎語法）

| 篇章 | 主題 |
|------|------|
| **基礎篇**（16 章） | 環境安裝、變數與資料型別、流程控制、函式、陣列與字串、指標基礎、運算子進階、進階指標、動態記憶體配置、自訂資料型態、檔案處理、前置處理器與模組化、鏈結串列實作、儲存類別與型別修飾詞、C99/C11 現代語法特性、goto/跨函式跳躍/可變參數 |

## 特色

- ✅ 完整中文教材（繁體中文）
- ✅ Python 完整課程（基礎→進階→實戰→演算法→解題技巧）
- ✅ Java 完整課程（基礎→物件導向→進階語法特色→Spring Boot 實務藍圖，共 25 章）
- ✅ C 語言基礎語法篇，快速掌握不同語言寫法
- ✅ 支援 Mermaid 圖表
- ✅ 基於 VitePress 建置，響應式設計
- ✅ 支援深色模式

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置靜態網站
npm run build

# 預覽建置結果
npm run preview
```

## 部署

推送至 `main` 分支即自動透過 GitHub Actions 部署到 GitHub Pages。
