# 🚀 實戰篇

歡迎來到 Python 實戰篇！本篇章將帶領你運用先前所學，解決真實世界的問題，建立完整的專案開發能力。

## 學習目標

完成本篇章後，你將能夠：

- ✅ 撰寫網路爬蟲擷取網頁資料
- ✅ 使用 SQLite / MySQL 進行資料庫操作
- ✅ 使用 FastAPI 建立 RESTful API
- ✅ 運用 Pandas 進行基本資料分析
- ✅ 撰寫自動化腳本提升工作效率
- ✅ 使用多執行緒處理並行任務
- ✅ 準備 ITS Python 國際認證考試

## 章節列表

### [01 - 網路爬蟲](./01-網路爬蟲)
學習使用 `requests` 與 `BeautifulSoup` 擷取網頁資料，處理動態內容與遵守爬蟲禮儀。

### [02 - 資料庫操作](./02-資料庫操作)
使用 `sqlite3` 模組與 SQLAlchemy ORM 進行資料庫的增刪改查操作。

### [03 - Web API 開發](./03-Web-API)
使用 FastAPI 框架建立現代化的 RESTful API，包含資料驗證與自動生成文件。

### [04 - 資料科學入門](./04-資料科學入門)
初步認識 NumPy 與 Pandas 的使用，進行資料載入、清理與基本分析。

### [05 - 自動化腳本](./05-自動化腳本)
撰寫實用的自動化腳本，涵蓋檔案整理、批次處理與排程任務。

### [06 - 執行緒與並行程式設計](./06-執行緒)
學習多執行緒程式設計，理解 GIL 的限制，並使用 Lock、Queue 等機制撰寫安全的並行程式。

### [07 - ITS Python 認證準備](./07-ITS認證考題)
針對 ITS Python 國際認證的重點整理與考古題練習，幫助你取得認證。

---

## 預備知識

本篇章假設你已熟悉 Python 基礎與進階概念。建議先完成前兩個篇章再開始。
