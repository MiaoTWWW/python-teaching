## 什麼是堆疊？

堆疊（Stack）是一種 **後進先出（LIFO, Last In First Out）** 的資料結構。就像疊盤子一樣，最後放上去的盤子最先被拿走。

### 基本操作

- **push** — 從頂端加入元素
- **pop** — 從頂端移除元素
- **peek / top** — 查看頂端元素
- **isEmpty** — 檢查是否為空

## Python 實作

```python
# 使用 list 實作堆疊（高效）
stack = []

# push
stack.append(1)
stack.append(2)
stack.append(3)
print(stack)  # [1, 2, 3]

# pop
top = stack.pop()  # 3
print(stack)       # [1, 2]

# peek
print(stack[-1])   # 2

# isEmpty
print(len(stack) == 0)  # False
```

## 應用 1：括號匹配

```python
def is_valid_parentheses(s):
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    for ch in s:
        if ch in mapping:
            if not stack or stack[-1] != mapping[ch]:
                return False
            stack.pop()
        else:
            stack.append(ch)
    return not stack

print(is_valid_parentheses("()[]{}"))  # True
print(is_valid_parentheses("([)]"))    # False
print(is_valid_parentheses("({[]})"))  # True
```

## 應用 2：中序轉後序 (Shunting Yard)

中序表達式 `A + B * C` → 後序 `A B C * +`

```python
def infix_to_postfix(expression):
    precedence = {'+': 1, '-': 1, '*': 2, '/': 2, '^': 3}
    stack = []
    output = []
    tokens = expression.split()
    for token in tokens:
        if token.isalnum():  # 運算元
            output.append(token)
        elif token == '(':
            stack.append(token)
        elif token == ')':
            while stack and stack[-1] != '(':
                output.append(stack.pop())
            stack.pop()  # 移除 '('
        else:  # 運算子
            while (stack and stack[-1] != '(' and
                   precedence.get(stack[-1], 0) >= precedence.get(token, 0)):
                output.append(stack.pop())
            stack.append(token)
    while stack:
        output.append(stack.pop())
    return ' '.join(output)

print(infix_to_postfix("A + B * C"))
# A B C * +

print(infix_to_postfix("( A + B ) * C"))
# A B + C *
```

## 應用 3：後序運算式求值

```python
def evaluate_postfix(expression):
    stack = []
    for token in expression.split():
        if token.isdigit():
            stack.append(int(token))
        else:
            b = stack.pop()
            a = stack.pop()
            if token == '+': stack.append(a + b)
            elif token == '-': stack.append(a - b)
            elif token == '*': stack.append(a * b)
            elif token == '/': stack.append(a / b)
    return stack[0]

print(evaluate_postfix("3 4 + 5 *"))  # (3+4)*5 = 35
```

## 應用 4：中序轉前序

```python
def infix_to_prefix(expression):
    # 反轉表達式，交換括號，轉後序，再反轉
    tokens = expression.split()
    reversed_exp = []
    for token in reversed(tokens):
        if token == '(':
            reversed_exp.append(')')
        elif token == ')':
            reversed_exp.append('(')
        else:
            reversed_exp.append(token)
    # 轉後序（但改成右結合優先）
    postfix = infix_to_postfix(' '.join(reversed_exp))
    return ' '.join(reversed(postfix.split()))
```
