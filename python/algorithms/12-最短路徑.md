## Dijkstra 演算法

用於**非負權重**圖形，找出單一起點到所有點的最短路徑。使用優先佇列（Min Heap）達到 O(E log V)。

```python
import heapq

def dijkstra(graph, start, n):
    """Dijkstra 最短路徑演算法
    graph: 鄰接串列 {節點: [(鄰居, 權重)]}
    start: 起點
    n: 節點數量
    回傳: dist（起點到各點的最短距離）
    """
    INF = float('inf')
    dist = [INF] * n
    dist[start] = 0
    pq = [(0, start)]  # (距離, 節點)
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]:
            continue  # 跳過過期項目
        for v, w in graph.get(u, []):
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(pq, (dist[v], v))
    return dist

# 範例
graph = {
    0: [(1, 4), (2, 1)],
    1: [(3, 1)],
    2: [(1, 2), (3, 5)],
    3: []
}
dist = dijkstra(graph, 0, 4)
print(dist)  # [0, 3, 1, 4]
# 0→0:0, 0→2→1:3, 0→2:1, 0→2→3:4 或 0→1→3:5 → 最短路徑 0→2→1→3 = 4
```

### 記錄路徑

```python
def dijkstra_with_path(graph, start, n):
    INF = float('inf')
    dist = [INF] * n
    prev = [-1] * n  # 前一個節點
    dist[start] = 0
    pq = [(0, start)]
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]:
            continue
        for v, w in graph.get(u, []):
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                prev[v] = u
                heapq.heappush(pq, (dist[v], v))
    # 重建路徑
    def get_path(target):
        path = []
        while target != -1:
            path.append(target)
            target = prev[target]
        return path[::-1]
    return dist, get_path
```

## Bellman-Ford 演算法

可處理**負權重**的邊，並能檢測負環路，時間複雜度 O(VE)。

```python
def bellman_ford(edges, n, start):
    """Bellman-Ford 演算法
    edges: [(u, v, w)] 邊的列表
    n: 節點數量
    start: 起點
    回傳: dist 或 None（有負環路）
    """
    INF = float('inf')
    dist = [INF] * n
    dist[start] = 0
    # 第 1~n-1 次鬆弛
    for _ in range(n - 1):
        for u, v, w in edges:
            if dist[u] != INF and dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
    # 第 n 次檢查負環路
    for u, v, w in edges:
        if dist[u] != INF and dist[u] + w < dist[v]:
            return None  # 存在負環路
    return dist

edges = [(0, 1, 4), (0, 2, 1), (2, 1, 2), (1, 3, 1), (2, 3, 5)]
dist = bellman_ford(edges, 4, 0)
print(dist)  # [0, 3, 1, 4]
```

## Floyd-Warshall 演算法

找出**所有點對**之間的最短路徑，時間複雜度 O(V³)。適合密集圖。

```python
def floyd_warshall(graph, n):
    """Floyd-Warshall 全點對最短路徑
    graph: n×n 矩陣，graph[i][j] = i→j 的權重（不存在為 INF）
    """
    INF = float('inf')
    dist = [row[:] for row in graph]
    # 初始化：自己到自己的距離為 0
    for i in range(n):
        dist[i][i] = 0
    # 動態規劃：逐步允許中間節點
    for k in range(n):
        for i in range(n):
            for j in range(n):
                if dist[i][k] != INF and dist[k][j] != INF:
                    dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
    return dist

INF = float('inf')
graph = [
    [0, 4, 1, INF],
    [INF, 0, INF, 1],
    [INF, 2, 0, 5],
    [INF, INF, INF, 0]
]
dist = floyd_warshall(graph, 4)
for row in dist:
    print(row)
# [0, 3, 1, 4]
# [inf, 0, inf, 1]
# [inf, 2, 0, 3]
# [inf, inf, inf, 0]
```

## 曼哈頓路徑 (Manhattan Path)

在網格中，只能沿著水平或垂直方向移動，兩點間的最短路徑長度為曼哈頓距離：

```python
def manhattan_distance(p1, p2):
    """計算兩點間的曼哈頓距離"""
    return abs(p1[0] - p2[0]) + abs(p1[1] - p2[1])

# 在網格中移動（僅限上下左右）
def min_path_in_grid(grid, start, end):
    """在網格中找最短路徑（BFS）"""
    rows, cols = len(grid), len(grid[0])
    from collections import deque
    queue = deque([start])
    visited = {start}
    steps = 0
    directions = [(-1,0), (1,0), (0,-1), (0,1)]
    while queue:
        for _ in range(len(queue)):
            r, c = queue.popleft()
            if (r, c) == end:
                return steps
            for dr, dc in directions:
                nr, nc = r+dr, c+dc
                if (0 <= nr < rows and 0 <= nc < cols and
                    grid[nr][nc] != 1 and (nr, nc) not in visited):
                    queue.append((nr, nc))
                    visited.add((nr, nc))
        steps += 1
    return -1
```

## 演算法比較

| 演算法 | 時間 | 空間 | 限制 | 用途 |
|--------|------|------|------|------|
| Dijkstra | O(E log V) | O(V) | 無負權 | 單源最短路 |
| Bellman-Ford | O(VE) | O(V) | 無負環路 | 含負權、檢測負環 |
| Floyd-Warshall | O(V³) | O(V²) | — | 全點對最短路 |
