# 01 - 程式設計基礎

> 🎯 **學習目標**：複習 Python 程式設計基礎，著重於比賽與檢定中常用的輸入輸出技巧與型別處理。

## 輸入與輸出

### 基本輸入

```python
# input() 一次讀取一行字串
name = input("請輸入姓名：")
print(f"你好，{name}！")
```

### 快速輸入 — 讀取多個數值

比賽中常需要一次讀取多個數值：

```python
# 一行包含多個整數
a, b, c = map(int, input().split())

# 讀取整行轉為整數串列
nums = list(map(int, input().split()))
```

### 大量資料 — sys.stdin.read

當輸入量很大（數萬行）時，`input()` 的速度不夠快：

```python
import sys

# 一次讀取所有輸入，再按空白分割
data = sys.stdin.read().split()
# 轉為整數串列
nums = list(map(int, data))
```

### 讀取多行直到 EOF

```python
import sys

for line in sys.stdin:
    line = line.strip()
    if not line:
        continue
    # 處理每一行
    print(line.upper())
```

## 型別轉換

```python
# 字串 → 整數
s = "123"
n = int(s)       # 123

# 字串 → 浮點數
f = float("3.14")  # 3.14

# 整數 → 二進位、八進位、十六進位字串
bin(42)   # '0b101010'
oct(42)   # '0o52'
hex(42)   # '0x2a'
```

### 常見陷阱

```python
# 字串包含非數字字元會拋錯
# int("12a") → ValueError

# 浮點數字串不能直接轉 int（需先轉 float）
# int("3.14") → ValueError
int(float("3.14"))  # 3

# 空白字元處理
s = "  hello  "
s.strip()  # "hello" — 移除前後空白
```

## 字串格式化

```python
# f-string（Python 3.6+）
name, score = "Alice", 95
print(f"{name} 得分：{score}")

# 格式化寬度與精度
pi = 3.1415926
print(f"{pi:.2f}")        # 3.14
print(f"{pi:10.3f}")      # "     3.142"（靠右對齊）
print(f"{pi:<10.3f}")     # "3.142     "（靠左對齊）

# 補零
print(f"{42:05d}")         # 00042
```

## 條件判斷與迴圈

```python
# 三元表達式
x = 10
result = "偶數" if x % 2 == 0 else "奇數"

# enumerate — 同時取索引與值
for i, val in enumerate(["a", "b", "c"]):
    print(i, val)

# zip — 同時迭代多個串列
for a, b in zip([1, 2, 3], [4, 5, 6]):
    print(a + b)
```

## 常用內建函式

```python
# 數學相關
abs(-5)          # 5
max(3, 7, 2)     # 7
min(3, 7, 2)     # 2
sum([1, 2, 3])   # 6
pow(2, 10)       # 1024（等同 2**10）
divmod(13, 4)    # (3, 1) — 商與餘數

# 排序
sorted([3, 1, 2])        # [1, 2, 3]
sorted([3, 1, 2], reverse=True)  # [3, 2, 1]

# lambda 搭配排序
data = [(1, "b"), (3, "a"), (2, "c")]
sorted(data, key=lambda x: x[1])  # [(3, 'a'), (1, 'b'), (2, 'c')]
```
