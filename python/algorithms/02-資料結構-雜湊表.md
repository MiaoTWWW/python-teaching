# 02 - 資料結構：雜湊表 (Hash Table)

> 🎯 **學習目標**：理解雜湊表的核心概念，掌握 Python 中 `dict` 與 `set` 的高效使用方式。

## 什麼是雜湊表？

雜湊表（Hash Table）是一種以 **鍵（Key）- 值（Value）** 配對儲存資料的結構，透過**雜湊函式（Hash Function）** 將鍵映射到陣列中的某個位置，達到平均 **O(1)** 的查詢、插入與刪除效能。

## Python 中的實作

### dict — 字典

```python
# 建立字典
d = {}
d = {"apple": 30, "banana": 20, "cherry": 50}

# 存取
print(d["apple"])    # 30
print(d.get("grape", 0))  # 0（避免 KeyError）

# 新增/修改
d["durian"] = 100

# 刪除
del d["banana"]
removed = d.pop("cherry")  # 回傳 50 並刪除

# 走訪
for key in d:
    print(key, d[key])

for key, val in d.items():
    print(key, val)
```

### set — 集合

```python
# 建立集合
s = {1, 2, 3, 4}
s = set([1, 2, 2, 3])  # {1, 2, 3} — 自動去重

# 基本操作
s.add(5)
s.remove(3)   # 元素不存在會拋 KeyError
s.discard(10) # 元素不存在不會拋錯

# 集合運算
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a & b   # 交集 {3, 4}
a | b   # 聯集 {1, 2, 3, 4, 5, 6}
a - b   # 差集 {1, 2}
a ^ b   # 對稱差集 {1, 2, 5, 6}
```

## 常見應用

### 1. 計數（Frequency Counter）

```python
# 計算每個字元出現次數
text = "hello world"
freq = {}
for ch in text:
    freq[ch] = freq.get(ch, 0) + 1
# {'h': 1, 'e': 1, 'l': 3, 'o': 2, ' ': 1, 'w': 1, 'r': 1, 'd': 1}

# 使用 Counter（更簡潔）
from collections import Counter
freq = Counter(text)
print(freq.most_common(3))  # [('l', 3), ('o', 2), ('h', 1)]
```

### 2. 去重（Deduplication）

```python
nums = [1, 2, 2, 3, 4, 4, 5]
unique = list(set(nums))       # [1, 2, 3, 4, 5]（順序不保證）
# 保留順序
unique_ordered = list(dict.fromkeys(nums))  # [1, 2, 3, 4, 5]
```

### 3. 兩數之和（Two Sum）

經典面試題：找出陣列中兩個數，使其和等於目標值。

```python
def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
```

### 4. 記憶化（Memoization）

```python
# 用 dict 暫存計算結果，避免重複計算
memo = {}
def fib(n):
    if n <= 1:
        return n
    if n not in memo:
        memo[n] = fib(n-1) + fib(n-2)
    return memo[n]
```

## 雜湊衝突與處理

當兩個不同的鍵產生相同的雜湊值時，稱為**碰撞（Collision）**。

### 鏈結法（Chaining）

同一個槽位用串列儲存多個元素。Python 的 `dict` 就是用這種方式處理碰撞。

### 開放定址法（Open Addressing）

碰撞時往後尋找下一個空位。Python 的 `set` 部分採用此策略。

## 時間複雜度

| 操作 | 平均 | 最差 |
|------|------|------|
| 查詢 | O(1) | O(n) |
| 插入 | O(1) | O(n) |
| 刪除 | O(1) | O(n) |

> 💡 `dict` 和 `set` 是 Python 中最強大的資料結構之一，善用它們可以大幅簡化程式碼並提升效能。
