## 什麼是遞迴？

遞迴（Recursion）是指一個函式**呼叫自己**來解決問題的方法。每個遞迴函式都需要：

1. **基本情況 (Base Case)** — 停止遞迴的條件
2. **遞迴情況 (Recursive Case)** — 將問題縮小後呼叫自己

```python
def countdown(n):
    """簡單的遞迴範例"""
    if n <= 0:        # 基本情況
        print("發射！")
        return
    print(n)          # 遞迴情況
    countdown(n - 1)

countdown(5)
# 5 4 3 2 1 發射！
```

### 階乘

```python
def factorial(n):
    """計算 n! = n × (n-1) × ... × 1"""
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120
```

def fibonacci(n):
    """第 n 個費波那契數"""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

# 最佳化：使用記憶化
from functools import lru_cache

@lru_cache(maxsize=None)
def fibonacci_memo(n):

## 河內塔 (Tower of Hanoi)

經典遞迴問題：將 n 個盤子從 A 柱移動到 C 柱，每次只能移動一個盤子，大盤子不能放在小盤子上。

```python
def hanoi(n, source, target, auxiliary):
    """河內塔解法
    n: 盤子數量
    source: 來源柱
    target: 目標柱
    auxiliary: 輔助柱
    """
    if n == 1:
        print(f"移動盤子 1：{source} → {target}")
        return
    hanoi(n - 1, source, auxiliary, target)
    print(f"移動盤子 {n}：{source} → {target}")
    hanoi(n - 1, auxiliary, target, source)

# 範例：3 個盤子
hanoi(3, 'A', 'C', 'B')
# 移動盤子 1：A → C
# 移動盤子 2：A → B
# 移動盤子 1：C → B
# 移動盤子 3：A → C
# 移動盤子 1：B → A
# 移動盤子 2：B → C
# 移動盤子 1：A → C
# 總共需要 2ⁿ - 1 = 7 步
```

## N-Queen 問題

在 N×N 的棋盤上放置 N 個皇后，使其不互相攻擊：

```python
def solve_n_queens(n):
    """N-Queen 問題求解"""
    result = []
    board = [['.'] * n for _ in range(n)]
    cols = set()      # 已佔用的列
    diag1 = set()     # 已佔用的主對角線（r + c）
    diag2 = set()     # 已佔用的副對角線（r - c）

    def backtrack(row):
        if row == n:
            result.append([''.join(r) for r in board])
            return
        for col in range(n):
            if col in cols or (row + col) in diag1 or (row - col) in diag2:
                continue
            # 放置皇后
            board[row][col] = 'Q'
            cols.add(col)
            diag1.add(row + col)
            diag2.add(row - col)
            backtrack(row + 1)
            # 移除皇后（回溯）
            board[row][col] = '.'
            cols.remove(col)
            diag1.remove(row + col)
            diag2.remove(row - col)

    backtrack(0)
    return result

solutions = solve_n_queens(4)
for sol in solutions:
    for row in sol:
        print(row)
    print()
# .Q..
# ...Q
# Q...
# ..Q.
#（4 皇后共有 2 組解）
```

## 數獨求解器

```python
def solve_sudoku(board):
    """使用回溯法解數獨
    board: 9×9 二維陣列，空格用 0 表示
    """
    def is_valid(board, row, col, num):
        # 檢查行
        for c in range(9):
            if board[row][c] == num:
                return False
        # 檢查列
        for r in range(9):
            if board[r][col] == num:
                return False
        # 檢查 3×3 方格
        box_r, box_c = row // 3 * 3, col // 3 * 3
        for r in range(box_r, box_r + 3):
            for c in range(box_c, box_c + 3):
                if board[r][c] == num:
                    return False
        return True

    def backtrack():
        for r in range(9):
            for c in range(9):
                if board[r][c] == 0:  # 空格
                    for num in range(1, 10):
                        if is_valid(board, r, c, num):
                            board[r][c] = num
                            if backtrack():
                                return True
                            board[r][c] = 0  # 回溯
                    return False
        return True  # 所有格子都填滿

    backtrack()
    return board

# 範例使用
board = [
    [5, 3, 0, 0, 7, 0, 0, 0, 0],
    [6, 0, 0, 1, 9, 5, 0, 0, 0],
    [0, 9, 8, 0, 0, 0, 0, 6, 0],
    [8, 0, 0, 0, 6, 0, 0, 0, 3],
    [4, 0, 0, 8, 0, 3, 0, 0, 1],
    [7, 0, 0, 0, 2, 0, 0, 0, 6],
    [0, 6, 0, 0, 0, 0, 2, 8, 0],
    [0, 0, 0, 4, 1, 9, 0, 0, 5],
    [0, 0, 0, 0, 8, 0, 0, 7, 9]
]

solve_sudoku(board)
for row in board:
    print(row)
```

## 遞迴 vs 迭代

| 特性 | 遞迴 | 迭代 |
|------|------|------|
| 程式碼 | 簡潔、接近數學定義 | 較長但直接 |
| 記憶體 | O(n) 呼叫堆疊 | O(1) 或可控 |
| 效能 | 較慢（函式呼叫開銷） | 較快 |
| 適合問題 | 樹、圖、分割問題 | 線性、迴圈問題 |

> 💡 遞迴的核心思維：**先假設子問題已經解決，再處理當前問題。** 很多看似複雜的問題（河內塔、N-Queen）用遞迴描述起來非常優雅。
