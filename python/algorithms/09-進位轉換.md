# 09 - 進位轉換

> 🎯 **學習目標**：掌握不同進位系統的轉換方法，並能用 Python 實作。

## 常見進位系統

| 進位 | 基數 | 數字範圍 | 前綴 |
|------|------|---------|------|
| 二進位 (Binary) | 2 | 0-1 | `0b` |
| 八進位 (Octal) | 8 | 0-7 | `0o` |
| 十進位 (Decimal) | 10 | 0-9 | — |
| 十六進位 (Hex) | 16 | 0-9, A-F | `0x` |

## Python 內建轉換

```python
# 十進位 → 其他進位字串
bin(42)   # '0b101010'
oct(42)   # '0o52'
hex(42)   # '0x2a'

# 去除前綴
bin(42)[2:]  # '101010'
hex(42)[2:]  # '2a'

# 任何進位字串 → 十進位
int('101010', 2)    # 42
int('52', 8)        # 42
int('2a', 16)       # 42
int('FF', 16)       # 255
```

## 手動實作轉換

### 十進位轉任意進位

```python
def decimal_to_base(n, base):
    """將十進位 n 轉為 base 進位字串"""
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    if n == 0:
        return "0"
    result = ""
    while n > 0:
        result = digits[n % base] + result
        n //= base
    return result

print(decimal_to_base(42, 2))   # "101010"
print(decimal_to_base(42, 8))   # "52"
print(decimal_to_base(42, 16))  # "2A"
print(decimal_to_base(255, 16)) # "FF"
```

### 任意進位轉十進位

```python
def base_to_decimal(s, base):
    """將 base 進位字串 s 轉為十進位"""
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    s = s.upper()
    result = 0
    for ch in s:
        result = result * base + digits.index(ch)
    return result

print(base_to_decimal("101010", 2))  # 42
print(base_to_decimal("2A", 16))     # 42
```

### 二進位與八/十六進位互轉（快捷法）

```python
# 二進位 → 十六進位：每 4 位一組
# 二進位 → 八進位：每 3 位一組

def bin_to_hex(bin_str):
    """二進位轉十六進位"""
    # 補齊位數使其為 4 的倍數
    padding = (4 - len(bin_str) % 4) % 4
    bin_str = "0" * padding + bin_str
    hex_str = ""
    mapping = "0123456789ABCDEF"
    for i in range(0, len(bin_str), 4):
        hex_str += mapping[int(bin_str[i:i+4], 2)]
    return hex_str
```

## 小數部分的轉換

```python
def decimal_fraction_to_base(fraction, base, precision=10):
    """將十進位小數 fraction（0~1）轉為 base 進位"""
    result = "."
    for _ in range(precision):
        fraction *= base
        digit = int(fraction)
        result += str(digit) if digit < 10 else chr(ord('A') + digit - 10)
        fraction -= digit
        if fraction == 0:
            break
    return result

print(decimal_fraction_to_base(0.375, 2))  # .011
print(decimal_fraction_to_base(0.1, 2))    # .0001100110...（循環）
```

## 進位轉換器

```python
def base_converter(num, from_base=10, to_base=10):
    """完整的進位轉換器"""
    # 先轉十進位
    if from_base != 10:
        num = base_to_decimal(num, from_base)
    # 再轉目標進位
    if to_base != 10:
        return decimal_to_base(num, to_base)
    return str(num)

# 範例：八進位轉十六進位
print(base_converter("52", 8, 16))  # "2A"
```
