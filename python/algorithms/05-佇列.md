# 05 - 佇列 (Queue)

> 🎯 **學習目標**：理解佇列的 FIFO 特性與 Python 實作方式，並能應用在 BFS 等問題中。

## 什麼是佇列？

佇列（Queue）是一種 **先進先出（FIFO, First In First Out）** 的資料結構。就像排隊買票一樣，先到的人先被服務。

### 基本操作

- **enqueue** — 從尾端加入元素
- **dequeue** — 從前端移除元素
- **peek / front** — 查看前端元素
- **isEmpty** — 檢查是否為空

## Python 實作

### 使用 collections.deque

`deque`（雙端佇列）是 Python 實作佇列的最佳選擇，兩端操作皆為 O(1)：

```python
from collections import deque

# 建立佇列
queue = deque()

# 加入元素（enqueue）
queue.append("A")
queue.append("B")
queue.append("C")
print(queue)  # deque(['A', 'B', 'C'])

# 取出元素（dequeue）
first = queue.popleft()  # 'A'
print(queue)             # deque(['B', 'C'])

# 查看前端元素
print(queue[0])  # 'B'

# 檢查是否為空
print(len(queue) == 0)  # False
```

### 使用 list（不建議）

```python
# 用 list 實作佇列 — pop(0) 是 O(n)，效率差！
queue = []
queue.append("A")     # enqueue
item = queue.pop(0)   # dequeue — 慢！
```

> 💡 除非資料量很小，否則不要用 `list` 實作佇列。`pop(0)` 需要移動所有元素。

## 環形佇列 (Circular Queue)

使用固定大小的陣列，透過頭尾指標循環利用空間：

```python
class CircularQueue:
    def __init__(self, size):
        self.size = size
        self.queue = [None] * size
        self.front = self.rear = -1

    def enqueue(self, item):
        if (self.rear + 1) % self.size == self.front:
            raise Exception("佇列已滿")
        elif self.front == -1:
            self.front = self.rear = 0
        else:
            self.rear = (self.rear + 1) % self.size
        self.queue[self.rear] = item

    def dequeue(self):
        if self.front == -1:
            raise Exception("佇列為空")
        item = self.queue[self.front]
        if self.front == self.rear:
            self.front = self.rear = -1
        else:
            self.front = (self.front + 1) % self.size
        return item

    def is_empty(self):
        return self.front == -1
```

## 應用：BFS（廣度優先搜尋）

BFS 是佇列最經典的應用之一，用於圖形或樹的層級走訪：

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    visited.add(start)
    while queue:
        node = queue.popleft()
        print(node, end=" ")
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

# 圖形表示（鄰接串列）
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}
bfs(graph, 'A')  # A B C D E F
```

## 時間複雜度

| 操作 | deque | list |
|------|-------|------|
| enqueue | O(1) | O(1) |
| dequeue | O(1) | O(n) |
| peek | O(1) | O(1) |
