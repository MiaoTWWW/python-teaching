# 07 - 排列組合

> 🎯 **學習目標**：理解排列與組合的數學概念，並能用 Python 實作與運用。

## 基本概念

- **排列 (Permutation)**：從 n 個相異物中取 k 個，考慮順序。記為 P(n, k)。
- **組合 (Combination)**：從 n 個相異物中取 k 個，不考慮順序。記為 C(n, k)。

### 數學公式

```
P(n, k) = n! / (n - k)!
C(n, k) = n! / (k! * (n - k)!)
```

## Python 的 itertools 實作

### 排列

```python
import itertools

items = ['A', 'B', 'C']

# 取 2 個的排列：P(3, 2) = 6
perms = list(itertools.permutations(items, 2))
print(perms)
# [('A', 'B'), ('A', 'C'), ('B', 'A'), ('B', 'C'), ('C', 'A'), ('C', 'B')]

# 取全部：P(3, 3) = 6
perms_full = list(itertools.permutations(items))
print(perms_full)
# [('A','B','C'), ('A','C','B'), ('B','A','C'), ('B','C','A'), ('C','A','B'), ('C','B','A')]
```

### 組合

```python
# 取 2 個的組合：C(3, 2) = 3
combs = list(itertools.combinations(items, 2))
print(combs)
# [('A', 'B'), ('A', 'C'), ('B', 'C')]

# 取 2 個的組合（可重複）
combs_with_r = list(itertools.combinations_with_replacement(items, 2))
print(combs_with_r)
# [('A','A'), ('A','B'), ('A','C'), ('B','B'), ('B','C'), ('C','C')]
```

### 笛卡爾積

```python
# 多組串列的笛卡爾積
for p in itertools.product([1, 2], ['a', 'b']):
    print(p)
# (1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')

# repeat 參數：自身乘積
for p in itertools.product([1, 2], repeat=3):
    print(p)
# (1,1,1), (1,1,2), ..., (2,2,2) 共 8 種
```

## 遞迴實作排列（回溯法）

```python
def permutations_recursive(arr):
    """使用回溯法產生所有排列"""
    result = []
    def backtrack(path, used):
        if len(path) == len(arr):
            result.append(path[:])
            return
        for i in range(len(arr)):
            if not used[i]:
                used[i] = True
                path.append(arr[i])
                backtrack(path, used)
                path.pop()
                used[i] = False
    backtrack([], [False] * len(arr))
    return result
```

## 數學運算實作

```python
import math

def nPr(n, r):
    """計算排列數 P(n, r)"""
    return math.perm(n, r)

def nCr(n, r):
    """計算組合數 C(n, r)"""
    return math.comb(n, r)

print(nPr(5, 2))  # 20
print(nCr(5, 2))  # 10

# 階乘
print(math.factorial(5))  # 120
```

## 應用練習

### 1. 產生所有子集合

```python
def all_subsets(arr):
    """產生所有子集合（共 2ⁿ 個）"""
    result = []
    for r in range(len(arr) + 1):
        for combo in itertools.combinations(arr, r):
            result.append(list(combo))
    return result

print(all_subsets([1, 2]))
# [[], [1], [2], [1, 2]]
```

### 2. 字謎（Anagram）檢查

```python
def is_anagram(s1, s2):
    """檢查兩個字串是否為字謎"""
    return sorted(s1) == sorted(s2)

# 或者使用 Counter
from collections import Counter
def is_anagram_counter(s1, s2):
    return Counter(s1) == Counter(s2)
```

### 3. 密碼暴力破解模擬

```python
import itertools
import string

def brute_force_generator(max_length=3):
    """產生所有可能密碼組合（小寫字母）"""
    chars = string.ascii_lowercase
    for length in range(1, max_length + 1):
        for combo in itertools.product(chars, repeat=length):
            yield ''.join(combo)
```

> 💡 排列組合在競賽中常用來窮舉小規模的解答空間，搭配 itertools 可以寫出非常簡潔的程式碼。
