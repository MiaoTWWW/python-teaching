# 13 - 動態規劃 (Dynamic Programming)

> 🎯 **學習目標**：理解動態規劃的核心概念，掌握常見 DP 問題的解題模式。

## 什麼是動態規劃？

動態規劃（DP）是將**大問題分解為重疊的子問題**，並將子問題的解**儲存起來**避免重複計算。

### 兩種實作方式

1. **Top-Down（記憶化搜尋）**：遞迴 + 備忘錄
2. **Bottom-Up（表格法）**：迭代填表

## 爬樓梯 (Climbing Stairs)

經典入門題：爬 n 階樓梯，每次可以走 1 或 2 階，有多少種方法？

```python
# Top-Down 記憶化
def climb_stairs_memo(n):
    memo = {}
    def dfs(i):
        if i <= 1:
            return 1
        if i not in memo:
            memo[i] = dfs(i-1) + dfs(i-2)
        return memo[i]
    return dfs(n)

# Bottom-Up
def climb_stairs(n):
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1], dp[2] = 1, 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 空間最佳化
def climb_stairs_optimized(n):
    if n <= 2:
        return n
    a, b = 1, 2
    for _ in range(3, n + 1):
        a, b = b, a + b
    return b
```

## 硬幣問題 (Coin Change)

給定不同面額的硬幣，求湊出目標金額的最少硬幣數：

```python
def coin_change(coins, amount):
    """最少硬幣數"""
    INF = float('inf')
    dp = [INF] * (amount + 1)
    dp[0] = 0
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)
    return dp[amount] if dp[amount] != INF else -1

print(coin_change([1, 2, 5], 11))  # 3（5+5+1）
print(coin_change([2], 3))         # -1（無法湊出）

# 計算組合數
def coin_change_ways(coins, amount):
    """湊出 amount 的方法數"""
    dp = [0] * (amount + 1)
    dp[0] = 1
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] += dp[i - coin]
    return dp[amount]

print(coin_change_ways([1, 2, 5], 5))  # 4（1+1+1+1+1, 1+1+1+2, 1+2+2, 5）
```

## 最長遞增子序列 (LIS)

```python
def length_of_lis(nums):
    """最長遞增子序列的長度 — O(n²)"""
    if not nums:
        return 0
    n = len(nums)
    dp = [1] * n
    for i in range(n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

# 最佳化 O(n log n) — 使用 patience sorting
def length_of_lis_optimized(nums):
    import bisect
    tails = []
    for num in nums:
        pos = bisect.bisect_left(tails, num)
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    return len(tails)

print(length_of_lis([10, 9, 2, 5, 3, 7, 101, 18]))  # 4（2, 5, 7, 101）
```

## 最長共用子序列 (LCS)

```python
def longest_common_subsequence(text1, text2):
    """最長共用子序列長度"""
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

# 回傳實際的 LCS 字串
def lcs_string(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    # 回溯找出字串
    i, j = m, n
    result = []
    while i > 0 and j > 0:
        if text1[i-1] == text2[j-1]:
            result.append(text1[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] > dp[i][j-1]:
            i -= 1
        else:
            j -= 1
    return ''.join(reversed(result))

print(longest_common_subsequence("abcde", "ace"))   # 3
print(lcs_string("abcde", "ace"))                   # "ace"
```

## 背包問題 (Knapsack)

### 0/1 背包

每個物品只能選 0 或 1 次：

```python
def knapsack_01(weights, values, capacity):
    """0/1 背包問題：最大價值"""
    n = len(weights)
    dp = [0] * (capacity + 1)
    for i in range(n):
        for w in range(capacity, weights[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[capacity]

weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
print(knapsack_01(weights, values, 5))  # 7（物品 0+1：3+4）
```

### 無限背包 (Unbounded)

每個物品可以選任意多次：

```python
def knapsack_unbounded(weights, values, capacity):
    """無限背包問題"""
    dp = [0] * (capacity + 1)
    for w in range(1, capacity + 1):
        for i in range(len(weights)):
            if weights[i] <= w:
                dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[capacity]

print(knapsack_unbounded([2, 3, 4], [3, 4, 5], 10))  # 15（3+3+4 或 2+2+2+4）
```

## 一棵小樹 (Tree DP)

```python
def tree_dp(graph, values, root, parent):
    """樹上 DP：選擇不相鄰節點的最大權重和
    graph: 鄰接串列
    values: 節點權重
    """
    # dp[0] = 不選當前節點, dp[1] = 選當前節點
    dp0, dp1 = 0, values[root]
    for child in graph[root]:
        if child == parent:
            continue
        c0, c1 = tree_dp(graph, values, child, root)
        dp0 += max(c0, c1)   # 不選當前節點，子節點可選可不選
        dp1 += c0             # 選當前節點，子節點只能不選
    return dp0, dp1
```

## DP 解題套路

1. **定義狀態**：`dp[i]` 或 `dp[i][j]` 代表什麼？
2. **找出轉移**：`dp[i]` 如何從之前的狀態推導？
3. **初始化**：`dp[0]` 或邊界條件是什麼？
4. **計算順序**：從前往後？從後往前？
5. **回傳答案**：通常是 `dp[n]` 或 `max(dp)`

> 💡 練習 DP 的關鍵：**先寫遞迴暴力解，再用 memoization 最佳化，最後改寫成 Bottom-Up。**
