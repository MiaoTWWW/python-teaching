# 🐍 基礎篇

歡迎來到 Python 基礎篇！本篇章將帶你從完全零基礎開始，建立起扎實的 Python 程式設計基本功。

## 學習目標

完成本篇章後，你將能夠：

- ✅ 安裝並設定 Python 開發環境
- ✅ 理解變數、資料型別與基本運算
- ✅ 使用條件判斷與迴圈控制程式流程
- ✅ 撰寫並呼叫函式
- ✅ 靈活運用 Python 內建的資料結構
- ✅ 處理與操作字串資料

## 章節列表

### [01 - 環境安裝與設定](./01-環境安裝)
學習如何安裝 Python、設定虛擬環境，以及選擇適合的開發工具。

### [02 - 變數與資料型別](./02-變數與資料型別)
了解 Python 中的變數概念、基本資料型別（整數、浮點數、字串、布林值）與型別轉換。

### [03 - 流程控制](./03-流程控制)
掌握 `if`/`elif`/`else` 條件判斷、`for` 與 `while` 迴圈，以及 `break`/`continue` 的使用時機。

### [04 - 函式](./04-函式)
學習如何定義與呼叫函式、參數傳遞、回傳值，以及變數作用域的概念。

### [05 - 資料結構](./05-資料結構)
深入探索 Python 的四大內建資料結構：列表（list）、元組（tuple）、集合（set）、字典（dict）。

### [06 - 字串處理](./06-字串處理)
學習字串的各種操作方法，包括切割、取代、格式化與正則表達式的基礎應用。

---

## 預備知識

本篇章 **不需要任何程式設計經驗**，只要熟悉基本的電腦操作即可開始。

> 💡 **小提醒**：程式設計最好的學習方式就是 **動手寫程式**！請務必跟著範例實際操作。
