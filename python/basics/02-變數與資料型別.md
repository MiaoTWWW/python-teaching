# 02 - 變數與資料型別

> 🎯 **學習目標**：理解 Python 中的變數概念、基本資料型別以及型別轉換。

## 變數（Variables）

在 Python 中，變數是用來**儲存資料**的容器。Python 是**動態型別**語言，不需要事先宣告變數的型別。

```python
# 變數指派
name = "Alice"      # 字串
age = 25             # 整數
height = 165.5      # 浮點數
is_student = True   # 布林值

# 輸出變數
print(name)          # Alice
print(age)           # 25
```

### 變數命名規則

| 規則 | 範例 |
|------|------|
| 只能包含字母、數字、底線 | `my_var`、`var1` |
| 不能以數字開頭 | ~~`1var`~~ ❌ |
| 區分大小寫 | `name` 和 `Name` 不同 |
| 不能使用保留字 | ~~`if`~~、~~`for`~~ ❌ |

**命名慣例（PEP 8）：**

```python
# ✅ 好的命名
user_name = "Bob"
total_score = 95
is_active = True

# ❌ 不好的命名
u = "Bob"          # 不明確
totalScore = 95    # 應使用底線命名法
```

## 基本資料型別

Python 有四種基本的資料型別：

### 整數（int）

```python
# 整數運算
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333... (除法永遠回傳浮點數)
print(a // b)  # 3 (整數除法)
print(a % b)   # 1 (取餘數)
print(a ** b)  # 1000 (次方)
```

### 浮點數（float）

```python
# 浮點數運算
pi = 3.14159
radius = 5.0

area = pi * radius ** 2
print(f"圓面積: {area:.2f}")  # 78.54
```

> ⚠️ **注意**：浮點數有精度問題，不要直接用 `==` 比較。

### 字串（str）

```python
# 字串建立
str1 = '單引號'
str2 = "雙引號"
str3 = """多行字串
可以換行
繼續寫"""

# 字串運算
greeting = "Hello" + " " + "World"  # 串接
repeat = "Ha" * 3                    # 重複 => "HaHaHa"

# f-string（格式化字串）
name = "Python"
version = 3.12
print(f"歡迎來到 {name} {version}")  # 推薦使用
```

### 布林值（bool）

```python
# 布林值
is_true = True
is_false = False

# 邏輯運算
print(True and False)   # False
print(True or False)    # True
print(not True)         # False

# 比較運算
print(5 > 3)     # True
print(5 == 3)    # False
print(5 != 3)    # True
```

## 型別轉換

```python
# 隱式轉換（自動）
result = 10 + 3.14    # int 自動轉為 float => 13.14

# 顯式轉換（手動）
# 字串轉數字
age_str = "25"
age_num = int(age_str)           # "25" -> 25
height_float = float("165.5")    # "165.5" -> 165.5

# 數字轉字串
score = 95
score_str = str(score)           # 95 -> "95"

# 轉換為布林值
print(bool(1))       # True
print(bool(0))       # False
print(bool(""))      # False
print(bool("abc"))   # True
```

## 型別查詢

```python
# 使用 type() 查詢資料型別
print(type(42))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type("Hello"))     # <class 'str'>
print(type(True))        # <class 'bool'>

# 使用 isinstance() 檢查型別
print(isinstance(42, int))         # True
print(isinstance("Hi", str))       # True
```

## None 型別

`None` 表示「沒有值」或「空」：

```python
result = None
print(result)           # None
print(result is None)   # True
```

## 重點整理

| 型別 | 類別名稱 | 範例 |
|------|---------|------|
| 整數 | `int` | `42`、`-10`、`0` |
| 浮點數 | `float` | `3.14`、`-0.5`、`1.0` |
| 字串 | `str` | `"Hello"`、`'Python'` |
| 布林值 | `bool` | `True`、`False` |
| 空值 | `NoneType` | `None` |
