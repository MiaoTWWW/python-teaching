# 02 - 變數與資料型別

> 🎯 **學習目標**：理解 Python 中的變數概念、基本資料型別以及型別轉換。

## 變數（Variables）

在 Python 中，變數是用來**儲存資料**的容器。Python 是**動態型別**語言，不需要事先宣告變數的型別。

```python
# 變數指派
name = "Alice"      # 字串
age = 25             # 整數
height = 165.5      # 浮點數
is_student = True   # 布林值

# 輸出變數
print(name)          # Alice
print(age)           # 25
```

### 變數命名規則

| 規則 | 範例 |
|------|------|
| 只能包含字母、數字、底線 | `my_var`、`var1` |
| 不能以數字開頭 | ~~`1var`~~ ❌ |
| 區分大小寫 | `name` 和 `Name` 不同 |
| 不能使用保留字 | ~~`if`~~、~~`for`~~ ❌ |

**命名慣例（PEP 8）：**

```python
# ✅ 好的命名
user_name = "Bob"
total_score = 95
is_active = True

# ❌ 不好的命名
u = "Bob"          # 不明確
totalScore = 95    # 應使用底線命名法
```

## 基本資料型別

Python 有四種基本的資料型別：

### 整數（int）

```python
# 整數運算
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333... (除法永遠回傳浮點數)
print(a // b)  # 3 (整數除法)
print(a % b)   # 1 (取餘數)
print(a ** b)  # 1000 (次方)
```

### 浮點數（float）

```python
# 浮點數運算
pi = 3.14159
radius = 5.0

area = pi * radius ** 2
print(f"圓面積: {area:.2f}")  # 78.54
```

> ⚠️ **注意**：浮點數有精度問題，不要直接用 `==` 比較。

### 浮點數的精度陷阱

電腦使用**二進位**儲存數字，但有些十進位的小數（如 0.1）無法用二進位精確表示，導致計算結果與預期不符：

```python
print(0.1 + 0.2)       # 0.30000000000000004（不是 0.3！）
print(0.1 + 0.2 == 0.3) # False（經典陷阱）
print(0.3)              # 0.3（但其實內部也不是精確的 0.3）
```

**常見問題與解法：**

```python
# ❌ 直接比較浮點數
if 0.1 + 0.2 == 0.3:       # 永遠不會成立
    print("相等")

# ✅ 解法 1：容差比較
epsilon = 1e-10
if abs((0.1 + 0.2) - 0.3) < epsilon:
    print("在誤差範圍內視為相等")

# ✅ 解法 2：使用 round() 四捨五入到指定位數
if round(0.1 + 0.2, 10) == round(0.3, 10):
    print("四捨五入後相等")

# ✅ 解法 3：使用 Decimal（精確十進位運算）
from decimal import Decimal
result = Decimal('0.1') + Decimal('0.2')
print(result)               # 0.3（精確！）
print(result == Decimal('0.3'))  # True
```

> 💡 **建議**：一般運算可使用 `round()` 或容差比較；需要**精確金錢計算**時，請使用 `Decimal`。

### 浮點數的進階運算

Python 提供多種方式處理浮點數的整數部分與進位：

```python
import math

num = 3.7

# ──── 取出整數部分 ────
print(int(num))              # 3（直接截斷小數，不進位）
print(num // 1)              # 3.0（整數除法，結果仍是 float）
print(math.trunc(num))       # 3（truncate，同 int()，截斷）

# ──── 無條件進位（Ceil） ────
print(math.ceil(3.1))        # 4（往上進位）
print(math.ceil(-3.1))       # -3（往「較大值」方向進位）

# ──── 無條件捨去（Floor） ────
print(math.floor(3.9))       # 3（往下捨去）
print(math.floor(-3.1))      # -4（往「較小值」方向捨去）

# ──── 四捨五入（Round） ────
print(round(3.5))            # 4
print(round(3.4))            # 3
print(round(3.14159, 2))     # 3.14（指定小數位數）

# ⚠️ round() 的銀行家捨入（四捨六入五成雙）
print(round(2.5))            # 2（不是 3！）
print(round(3.5))            # 4
```

> 💡 **總結**：`int()` 直接砍掉小數、`math.floor()` 往下捨、`math.ceil()` 往上進、`round()` 四捨五入（注意銀行家捨入）。

### 字串（str）

```python
# 字串建立
str1 = '單引號'
str2 = "雙引號"
str3 = """多行字串
可以換行
繼續寫"""

# 字串運算
greeting = "Hello" + " " + "World"  # 串接
repeat = "Ha" * 3                    # 重複 => "HaHaHa"

# f-string（格式化字串）
name = "Python"
version = 3.12
print(f"歡迎來到 {name} {version}")  # 推薦使用
```

### 布林值（bool）

```python
# 布林值
is_true = True
is_false = False

# 邏輯運算
print(True and False)   # False
print(True or False)    # True
print(not True)         # False

# 比較運算
print(5 > 3)     # True
print(5 == 3)    # False
print(5 != 3)    # True
```

## 型別轉換

```python
# 隱式轉換（自動）
result = 10 + 3.14    # int 自動轉為 float => 13.14

# 顯式轉換（手動）
# 字串轉數字
age_str = "25"
age_num = int(age_str)           # "25" -> 25
height_float = float("165.5")    # "165.5" -> 165.5

# 數字轉字串
score = 95
score_str = str(score)           # 95 -> "95"

# 轉換為布林值
print(bool(1))       # True
print(bool(0))       # False
print(bool(""))      # False
print(bool("abc"))   # True
```

## 型別查詢

```python
# 使用 type() 查詢資料型別
print(type(42))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type("Hello"))     # <class 'str'>
print(type(True))        # <class 'bool'>

# 使用 isinstance() 檢查型別
print(isinstance(42, int))         # True
print(isinstance("Hi", str))       # True
```

## None 型別

`None` 表示「沒有值」或「空」：

```python
result = None
print(result)           # None
print(result is None)   # True
```

## 重點整理

| 型別 | 類別名稱 | 範例 |
|------|---------|------|
| 整數 | `int` | `42`、`-10`、`0` |
| 浮點數 | `float` | `3.14`、`-0.5`、`1.0` |
| 字串 | `str` | `"Hello"`、`'Python'` |
| 布林值 | `bool` | `True`、`False` |
| 空值 | `NoneType` | `None` |
