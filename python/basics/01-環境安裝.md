# 01 - 環境安裝與設定

> 🎯 **學習目標**：在本章中，你將學習如何安裝 Python、設定開發環境，並執行你的第一行程式碼。

## 什麼是 Python？

Python 是一種**高階、直譯式**的程式語言，由 Guido van Rossum 於 1991 年創造。它以簡潔易讀的語法聞名，被廣泛應用於：

- 🌐 網路開發
- 📊 資料科學與機器學習
- 🤖 自動化腳本
- 🎮 遊戲開發
- 🔬 科學計算

## 安裝 Python

### Windows 系統

1. 前往 [Python 官方網站](https://www.python.org/downloads/)
2. 點擊下載最新版本的 Python（建議 3.11 以上）
3. 執行安裝程式，**務必勾選** ✅ `Add Python to PATH`
4. 點擊 `Install Now` 完成安裝

![Python 安裝畫面](https://docs.python.org/zh-tw/3/_images/win_installer.png)

### 驗證安裝

開啟終端機（命令提示字元），輸入以下指令：

```bash
python --version
```

如果看到類似以下的輸出，表示安裝成功：

```
Python 3.12.0
```

## 開發環境選擇

你可以選擇以下任一方式來撰寫 Python 程式：

### 1️⃣ 直譯器互動模式

直接在終端機輸入 `python` 進入互動模式：

```bash
python
>>> print("Hello, Python!")
Hello, Python!
>>> exit()
```

### 2️⃣ VS Code（建議）

Visual Studio Code 是目前最受歡迎的 Python 開發工具。

**安裝步驟：**

1. 下載 [VS Code](https://code.visualstudio.com/)
2. 安裝 **Python 擴充套件**（由 Microsoft 提供）
3. 建立一個 `.py` 檔案，開始撰寫程式！

### 3️⃣ PyCharm

JetBrains 出品的專業 Python IDE，分為社群版（免費）與專業版。

## 虛擬環境

虛擬環境可以讓每個專案擁有獨立的套件環境，避免套件衝突。

### 使用 venv 建立虛擬環境

```bash
# 建立虛擬環境
python -m venv myenv

# 啟動虛擬環境（Windows）
myenv\Scripts\activate

# 啟動虛擬環境（macOS/Linux）
source myenv/bin/activate

# 離開虛擬環境
deactivate
```

### 使用 pip 安裝套件

```bash
# 安裝套件
pip install requests

# 安裝特定版本
pip install requests==2.31.0

# 列出已安裝套件
pip list

# 匯出套件列表
pip freeze > requirements.txt

# 從套件列表安裝
pip install -r requirements.txt
```

## 你的第一支 Python 程式

建立一個 `hello.py` 檔案：

```python
# hello.py
print("Hello, Python!")
print("歡迎來到 Python 程式設計的世界！")

# 計算並輸出結果
name = "Python"
version = 3.12
print(f"你正在學習 {name} {version}")
```

執行方式：

```bash
python hello.py
```

輸出：

```
Hello, Python!
歡迎來到 Python 程式設計的世界！
你正在學習 Python 3.12
```

## 🏆 經典練習題

### 環境設定 🟢 簡易

1. **安裝驗證**：安裝 Python 3.11+ 後，在終端機執行指令確認版本
   ```bash
   # 應該要看到 Python 3.x.x
   ```
   💡 提示：使用 `python --version` 指令

2. **PATH 設定**：安裝 Python 時忘記勾選「Add Python to PATH」，請手動加入
   💡 提示：Windows 在「系統環境變數」中編輯 PATH

3. **多重版本管理**：電腦同時安裝了 Python 3.11 和 3.12，如何分別使用？
   💡 提示：Windows 用 `py` 啟動器，macOS/Linux 用 `python3.11`、`python3.12`

### REPL 操作 🟢 簡易

4. **互動式計算機**：在 Python REPL 中計算 `(25 + 17) * 3 / 2` 的結果
   ```python
   # 預期輸出：63.0
   ```
   💡 提示：直接在終端機輸入 `python` 進入 REPL

5. **REPL 歷史**：如何在 REPL 中快速叫用上一次輸入的指令？
   💡 提示：方向鍵 ↑

6. **離開 REPL**：在 REPL 中輸入什麼可以回到終端機？
   💡 提示：`exit()` 或 Ctrl+Z（Windows）/ Ctrl+D（macOS/Linux）

### pip 套件管理 🟡 中等

7. **安裝套件**：使用 pip 安裝 `requests` 套件，並驗證安裝成功
   ```bash
   pip install requests
   ```
   💡 提示：安裝後在 REPL 輸入 `import requests` 確認沒有錯誤

8. **匯出依賴**：將目前虛擬環境中所有已安裝的套件匯出到檔案
   💡 提示：使用 `pip freeze > requirements.txt`

9. **版本衝突**：專案 A 需要 `flask==2.0`，專案 B 需要 `flask==3.0`，如何解決？
   💡 提示：為每個專案建立獨立虛擬環境

### 基礎輸入輸出 🟢 簡易

10. **自我介紹**：寫一支程式，用變數儲存姓名、年齡、城市，然後用 print 輸出自我介紹
    ```python
    # 預期輸出：
    # 大家好，我是 XXX，今年 XX 歲，住在 XXX。
    ```
    💡 提示：用 f-string 格式化輸出

11. **簡易問答**：寫一支程式，詢問使用者「你最喜歡的程式語言是什麼？」，然後輸出回應
    💡 提示：使用 `input()` 函式

12. **格式化輸出**：印出一個 3×3 的乘法表區塊，讓數字用 `:4d` 對齊
    💡 提示：在 f-string 中使用 `{value:4d}` 指定寬度

### 開發環境整合 🔴 困難

13. **VS Code 偵錯**：在 VS Code 中設定中斷點，逐步執行一支包含 for 迴圈的程式，觀察變數變化
    💡 提示：點擊行號左側設定中斷點，按 F5 啟動偵錯

14. **一鍵執行**：設定 VS Code 的 launch.json，讓你可以按 F5 一鍵執行 currently 開啟的 Python 檔案
    💡 提示：在 Run and Debug 面板建立 Python 啟動設定
