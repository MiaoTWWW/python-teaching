# 01 - 環境安裝與設定

> 🎯 **學習目標**：在本章中，你將學習如何安裝 Python、設定開發環境，並執行你的第一行程式碼。

## 什麼是 Python？

Python 是一種**高階、直譯式**的程式語言，由 Guido van Rossum 於 1991 年創造。它以簡潔易讀的語法聞名，被廣泛應用於：

- 🌐 網路開發
- 📊 資料科學與機器學習
- 🤖 自動化腳本
- 🎮 遊戲開發
- 🔬 科學計算

## 安裝 Python

### Windows 系統

1. 前往 [Python 官方網站](https://www.python.org/downloads/)
2. 點擊下載最新版本的 Python（建議 3.11 以上）
3. 執行安裝程式，**務必勾選** ✅ `Add Python to PATH`
4. 點擊 `Install Now` 完成安裝

![Python 安裝畫面](https://docs.python.org/zh-tw/3/_images/win_installer.png)

### 驗證安裝

開啟終端機（命令提示字元），輸入以下指令：

```bash
python --version
```

如果看到類似以下的輸出，表示安裝成功：

```
Python 3.12.0
```

## 開發環境選擇

你可以選擇以下任一方式來撰寫 Python 程式：

### 1️⃣ 直譯器互動模式

直接在終端機輸入 `python` 進入互動模式：

```bash
python
>>> print("Hello, Python!")
Hello, Python!
>>> exit()
```

### 2️⃣ VS Code（建議）

Visual Studio Code 是目前最受歡迎的 Python 開發工具。

**安裝步驟：**

1. 下載 [VS Code](https://code.visualstudio.com/)
2. 安裝 **Python 擴充套件**（由 Microsoft 提供）
3. 建立一個 `.py` 檔案，開始撰寫程式！

### 3️⃣ PyCharm

JetBrains 出品的專業 Python IDE，分為社群版（免費）與專業版。

## 虛擬環境

虛擬環境可以讓每個專案擁有獨立的套件環境，避免套件衝突。

### 使用 venv 建立虛擬環境

```bash
# 建立虛擬環境
python -m venv myenv

# 啟動虛擬環境（Windows）
myenv\Scripts\activate

# 啟動虛擬環境（macOS/Linux）
source myenv/bin/activate

# 離開虛擬環境
deactivate
```

### 使用 pip 安裝套件

```bash
# 安裝套件
pip install requests

# 安裝特定版本
pip install requests==2.31.0

# 列出已安裝套件
pip list

# 匯出套件列表
pip freeze > requirements.txt

# 從套件列表安裝
pip install -r requirements.txt
```

## 你的第一支 Python 程式

建立一個 `hello.py` 檔案：

```python
# hello.py
print("Hello, Python!")
print("歡迎來到 Python 程式設計的世界！")

# 計算並輸出結果
name = "Python"
version = 3.12
print(f"你正在學習 {name} {version}")
```

執行方式：

```bash
python hello.py
```

輸出：

```
Hello, Python!
歡迎來到 Python 程式設計的世界！
你正在學習 Python 3.12
```

## 重點整理

| 項目 | 說明 |
|------|------|
| Python 版本 | 建議使用 3.11 以上版本 |
| 開發工具 | VS Code + Python 擴充套件 |
| 虛擬環境 | 每個專案使用獨立的虛擬環境 |
| 套件管理 | 使用 `pip` 管理第三方套件 |
