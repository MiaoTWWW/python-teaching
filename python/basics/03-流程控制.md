# 03 - 流程控制

> 🎯 **學習目標**：掌握條件判斷與迴圈控制，讓程式能夠根據不同情況做出決策。

## 條件判斷（if / elif / else）

### 基本 if 語法

```python
age = 18

if age >= 18:
    print("你已經成年了！")
    print("可以投票。")
```

### if / else 語法

```python
score = 85

if score >= 60:
    print("及格！")
else:
    print("不及格，再接再厲！")
```

### if / elif / else 語法

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"成績等級: {grade}")
```

### 巢狀條件判斷

```python
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("可以進入")
    else:
        print("需要出示證件")
else:
    print("未滿 18 歲，禁止進入")
```

### 三元運算式

```python
# 一般寫法
age = 20
status = ""
if age >= 18:
    status = "成年"
else:
    status = "未成年"

# 三元運算式（一行搞定）
status = "成年" if age >= 18 else "未成年"
```

## 邏輯運算子

```python
# and（且）：兩個條件都為 True
age = 25
has_license = True
if age >= 18 and has_license:
    print("可以開車")

# or（或）：至少一個條件為 True
is_weekend = True
is_holiday = False
if is_weekend or is_holiday:
    print("放假囉！")

# not（反向）
is_rainy = False
if not is_rainy:
    print("今天不會下雨")
```

## 成員與身份運算子

```python
# in 成員運算子
fruits = ["蘋果", "香蕉", "橘子"]
print("蘋果" in fruits)      # True
print("葡萄" in fruits)      # False

# is 身份運算子（檢查是否為同一個物件）
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a is c)     # True（同一個物件）
print(a is b)     # False（不同物件，但內容相同）
print(a == b)     # True（內容相同）
```

## for 迴圈

### 基本 for 迴圈

```python
# 遍歷列表
fruits = ["蘋果", "香蕉", "橘子"]
for fruit in fruits:
    print(f"我喜歡吃 {fruit}")

# 使用 range()
for i in range(5):        # 0, 1, 2, 3, 4
    print(i)

for i in range(2, 8):     # 2, 3, 4, 5, 6, 7
    print(i)

for i in range(0, 10, 2): # 0, 2, 4, 6, 8（步長為 2）
    print(i)
```

### enumerate：同時取得索引與值

```python
fruits = ["蘋果", "香蕉", "橘子"]
for index, fruit in enumerate(fruits):
    print(f"第 {index + 1} 個水果是 {fruit}")
# 輸出：
# 第 1 個水果是 蘋果
# 第 2 個水果是 香蕉
# 第 3 個水果是 橘子
```

### 遍歷字典

```python
student = {
    "name": "Alice",
    "age": 20,
    "score": 95
}

# 遍歷鍵
for key in student:
    print(key)

# 遍歷值
for value in student.values():
    print(value)

# 同時遍歷鍵與值
for key, value in student.items():
    print(f"{key}: {value}")
```

## while 迴圈

```python
# 基本 while 迴圈
count = 0
while count < 5:
    print(f"計數: {count}")
    count += 1

# 無限迴圈（需要手動 break）
while True:
    user_input = input("輸入 'q' 離開: ")
    if user_input == 'q':
        print("再見！")
        break
```

## break 與 continue

```python
# break：跳出整個迴圈
for i in range(10):
    if i == 5:
        break
    print(i, end=" ")
# 輸出: 0 1 2 3 4

# continue：跳過本次迭代
for i in range(10):
    if i % 2 == 0:
        continue      # 跳過偶數
    print(i, end=" ")
# 輸出: 1 3 5 7 9
```

## 實用技巧

### 使用 else 搭配迴圈

```python
# for-else：當迴圈正常結束（沒有 break）時執行 else
for i in range(5):
    if i == 10:
        break
else:
    print("迴圈正常結束")  # 會執行

# while-else
count = 0
while count < 3:
    print(count)
    count += 1
else:
    print("while 迴圈結束")  # 會執行
```

### 列表推導式（List Comprehension）

```python
# 傳統寫法
squares = []
for x in range(10):
    squares.append(x ** 2)

# 列表推導式（更簡潔）
squares = [x ** 2 for x in range(10)]

# 加入條件
even_squares = [x ** 2 for x in range(10) if x % 2 == 0]
print(even_squares)  # [0, 4, 16, 36, 64]
```

## 重點整理

| 控制結構 | 用途 | 關鍵字 |
|---------|------|--------|
| 條件判斷 | 根據條件執行不同程式碼 | `if`、`elif`、`else` |
| for 迴圈 | 遍歷可迭代物件 | `for`、`in` |
| while 迴圈 | 條件成立時重複執行 | `while` |
| 跳躍控制 | 中斷或跳過迭代 | `break`、`continue` |
