## 字串常用方法

### 大小寫轉換

```python
text = "Hello, Python!"

print(text.upper())          # HELLO, PYTHON!
print(text.lower())          # hello, python!
print(text.title())          # Hello, Python!
print(text.capitalize())     # Hello, python!
print(text.swapcase())       # hELLO, pYTHON!
```

### 搜尋與取代

```python
text = "Hello, Python! Python 很有趣。"

# 搜尋
print(text.find("Python"))      # 7（回傳第一個出現的索引）
print(text.find("Java"))        # -1（找不到）
print(text.index("Python"))     # 7（找不到會拋錯）
print(text.count("Python"))     # 2（計算出現次數）
print(text.startswith("Hello")) # True
print(text.endswith("趣。"))    # True

# 取代
print(text.replace("Python", "Java"))
# Hello, Java! Java 很有趣。
print(text.replace("Python", "Java", 1))  # 只取代第一個
```

### 分割與連接

```python
# split：字串 → 列表
text = "蘋果,香蕉,橘子,葡萄"
fruits = text.split(",")
print(fruits)  # ['蘋果', '香蕉', '橘子', '葡萄']

# rsplit：從右邊開始分割
text = "a/b/c/d"
print(text.rsplit("/", 2))  # ['a/b', 'c', 'd']

# splitlines：按換行分割
text = "第一行\n第二行\n第三行"
print(text.splitlines())  # ['第一行', '第二行', '第三行']

# join：列表 → 字串
fruits = ["蘋果", "香蕉", "橘子"]
print("、".join(fruits))  # 蘋果、香蕉、橘子
```

### 去除空白

```python
text = "  Hello, World!  \n"

print(text.strip())      # "Hello, World!"（去除前後空白）
print(text.lstrip())     # "Hello, World!  \n"（去除左側）
print(text.rstrip())     # "  Hello, World!"（去除右側）

# 自訂去除字元
text = "---Hello---"
print(text.strip("-"))   # "Hello"
```

### 檢查字串內容

```python
print("12345".isdigit())       # True（是否全是數字）
print("abc".isalpha())         # True（是否全是字母）
print("abc123".isalnum())      # True（是否全是字母或數字）
print("Hello".isupper())       # False（是否大寫）
print("hello".islower())       # True（是否小寫）
print("   ".isspace())         # True（是否全是空白）
print("Hello World".istitle()) # True（是否標題格式）
```

## 字串格式化

### f-string（Python 3.6+，推薦）

```python
name = "Alice"
age = 25
score = 85.5678

# 基本使用
print(f"姓名: {name}, 年齡: {age}")

# 格式化數字
print(f"分數: {score:.1f}")    # 85.6（小數點後一位）
print(f"分數: {score:.2f}")    # 85.57（小數點後兩位）

# 補零與對齊
print(f"{name:>10}")   # 右對齊（寬度 10）
print(f"{name:<10}")   # 左對齊（寬度 10）
print(f"{name:^10}")   # 置中（寬度 10）
print(f"{42:05d}")     # 補零: 00042

# 百分比
rate = 0.8567
print(f"完成率: {rate:.1%}")  # 85.7%

# 運算式
print(f"總和: {3 + 5}")
print(f"大寫: {name.upper()}")
```

### format() 方法

```python
# 位置參數
print("{} 今年 {} 歲".format("Alice", 25))

# 索引
print("{1} 今年 {0} 歲".format(25, "Alice"))

# 關鍵字參數
print("{name} 今年 {age} 歲".format(name="Alice", age=25))
```

## 正則表達式基礎

正則表達式（Regular Expression）是處理字串的強大工具。

```python
import re

text = "我的 email 是 alice@example.com，電話是 0912-345-678"

# 搜尋 email 模式
pattern = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b'
email = re.search(pattern, text)
if email:
    print(f"找到 Email: {email.group()}")

# 尋找所有數字
numbers = re.findall(r'\d+', text)
print(f"找到的數字: {numbers}")

# 取代
text = "今天是 2024/01/15"
result = re.sub(r'(\d{4})/(\d{2})/(\d{2})', r'\1年\2月\3日', text)
print(result)  # 今天是 2024年01月15日
```

### 常用正則表達式模式

| 模式 | 說明 | 範例 |
|------|------|------|
| `\d` | 數字 | `\d{4}` 匹配四位數字 |
| `\w` | 字母、數字、底線 | `\w+` 匹配一個單字 |
| `\s` | 空白字元 | `\s` 匹配空白、tab、換行 |
| `.` | 任意字元（除換行） | `h.t` 匹配 hat、hot |
| `*` | 零次或多次 | `ab*c` 匹配 ac、abc、abbc |
| `+` | 一次或多次 | `ab+c` 匹配 abc、abbc |
| `?` | 零次或一次 | `ab?c` 匹配 ac、abc |
| `^` | 開頭 | `^Hello` 以 Hello 開頭 |
| `$` | 結尾 | `end$` 以 end 結尾 |

## 字串編碼

```python
# 編碼與解碼
text = "Python 程式設計"
encoded = text.encode("utf-8")
print(encoded)  # b'Python \xe7\xa8\x8b...'

decoded = encoded.decode("utf-8")
print(decoded)  # Python 程式設計

# 跳脫字元
print("Hello\nWorld")     # 換行
print("Tab\there")        # Tab
print("反斜線: \\")       # 反斜線
print("雙引號: \"")       # 雙引號
```

## 🏆 經典練習題

### 字串索引與切片 🟢 簡易 | 🟡 中等

1. **取子字串**：從 `s = "Hello, Python!"` 中用切片取出 `"Python"`
   💡 提示：`s[7:13]`

2. **字串反轉**：一行反轉字串 `"abcdef"`
   💡 提示：`s[::-1]`

3. **間隔取值**：從 `"HelloWorld"` 中每隔兩個字元取一個
   💡 提示：`s[::3]`

4. **字串旋轉**：將 `"Python"` 向右旋轉 2 個字元 → `"onPyth"`
   💡 提示：`s[-2:] + s[:-2]`

### 字串 methods 🟢 簡易 | 🟡 中等

5. **單字計數**：給定一段英文文章，用 `split()` 計算單字總數
   💡 提示：`len(text.split())`

6. **CSV 解析**：字串 `"Alice,85,92,78"`，用 `split(",")` 解析為名稱和分數列表
   💡 提示：使用 `split()` 並指定分隔符

7. **去除多餘空白**：將 `"   Hello   World   "` 整理為 `"Hello World"`
   💡 提示：`strip()` 去除前後空白，`split()` 整理中間

8. **字串取代**：將 `"I like Python. Python is great!"` 中的所有 "Python" 取代為 "Java"
   💡 提示：`replace("Python", "Java")`

9. **檔名處理**：給定檔名 `"document.txt"`，取出主檔名（document）和副檔名（txt）
   💡 提示：`rsplit(".", 1)`

10. **字串檢查**：寫一個函式 `is_valid_username(s)` 檢查使用者名稱是否只包含字母、數字和底線
    💡 提示：`s.isalnum()` 或配合自訂條件

### 格式化輸出 🟢 簡易 | 🟡 中等

11. **金額格式**：將 `1234567` 格式化為 `"1,234,567"`
    💡 提示：`f"{num:,}"`

12. **報價單**：用 f-string 印出一個對齊的訂單表格
    ```python
    # 商品        單價   數量   小計
    # 蘋果        30      5    150
    # 香蕉        20      3     60
    ```
    💡 提示：指定欄寬 `{item:<10} {price:>5} {qty:>3} {subtotal:>6}`

13. **填補字串**：將 `"42"` 格式化為 `"0042"`（補滿 4 位）
    💡 提示：`f"{s:0>4}"` 或 `zfill(4)`

### 正則表達式入門 🟡 中等 | 🔴 困難

14. **Email 驗證**：寫一個正則表達式檢查字串是否為有效的 Email（含 @ 和 .）
    ```python
    import re
    pattern = r"^[\w.]+@\w+\.\w+$"
    ```
    💡 提示：`re.match()` 回傳 None 表示不符合

15. **電話號碼提取**：從一段文字中提取所有台灣手機號碼（09xx-xxx-xxx 格式）
    💡 提示：`re.findall(r"09\d{2}-\d{3}-\d{3}", text)`

16. **URL 解析**：從 `"Visit us at https://example.com/path?q=1"` 中提取 domain 部分
    💡 提示：`re.search(r"https?://([^/]+)", url)`

### 字串編碼處理 🟡 中等

17. **Unicode 轉換**：將字串 `"Hello"` 轉為 bytes 再轉回來
    💡 提示：`"Hello".encode("utf-8")` → `.decode("utf-8")`

18. **跳脫字元**：用一個字串表示路徑 `C:\Users\Name\file.txt`，注意跳脫問題
    💡 提示：用 raw string `r"C:\Users\Name\file.txt"` 或雙反斜線

### 文字分析實戰 🔴 困難

19. **回文檢查**：寫一個函式判斷字串是否為回文（忽略大小寫和非字母數字）
    ```python
    print(is_palindrome("A man, a plan, a canal: Panama"))  # True
    ```
    💡 提示：用 `re.sub(r"[^a-zA-Z0-9]", "", s).lower()` 清理字串

20. **字頻分析**：讀取一段文字，輸出出現頻率最高的前 5 個單字及其次數
    💡 提示：用 split 分詞 → Counter 統計 → most_common(5)
