# 04 - 檔案 I/O

> 🎯 **學習目標**：學習如何在 Python 中讀寫檔案，處理各種資料格式。

## 開啟與關閉檔案

### 基本檔案操作

```python
# 開啟檔案
file = open("example.txt", "r", encoding="utf-8")
content = file.read()
file.close()  # 記得關閉！
```

### 使用 with 語句（建議）

```python
# with 會自動關閉檔案
with open("example.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
# 離開 with 區塊後，檔案會自動關閉
```

## 檔案模式

| 模式 | 說明 |
|------|------|
| `"r"` | 讀取（預設）。檔案必須存在 |
| `"w"` | 寫入。會覆蓋已存在的檔案 |
| `"a"` | 附加。在檔案末尾新增內容 |
| `"x"` | 獨佔建立。檔案已存在則失敗 |
| `"b"` | 二進位模式（可與其他模式組合） |
| `"+"` | 讀寫模式（可與其他模式組合） |

```python
# 各種模式範例
with open("read.txt", "r", encoding="utf-8") as f:
    pass

with open("write.txt", "w", encoding="utf-8") as f:
    f.write("新內容")

with open("append.txt", "a", encoding="utf-8") as f:
    f.write("附加內容\n")

with open("new.txt", "x", encoding="utf-8") as f:
    f.write("獨佔建立")
```

## 讀取檔案

### 讀取方法

```python
# read() - 讀取全部內容
with open("sample.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# readline() - 逐行讀取
with open("sample.txt", "r", encoding="utf-8") as f:
    line = f.readline()
    while line:
        print(line, end="")
        line = f.readline()

# readlines() - 讀取所有行到列表
with open("sample.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    print(f"共有 {len(lines)} 行")

# 最常用的方式：直接迭代
with open("sample.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line, end="")
```

## 寫入檔案

```python
# write() - 寫入字串
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("第一行\n")
    f.write("第二行\n")

# writelines() - 寫入字串列表
lines = ["第一行\n", "第二行\n", "第三行\n"]
with open("output.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)

# print() 也可以寫入檔案
with open("output.txt", "w", encoding="utf-8") as f:
    print("這是第一行", file=f)
    print("這是第二行", file=f)
```

## CSV 檔案處理

```python
import csv

# 讀取 CSV
with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    header = next(reader)  # 跳過標題列
    for row in reader:
        name, age, score = row
        print(f"{name}: {age} 歲, 成績 {score}")

# 使用 DictReader（更方便）
with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(f"{row['name']}: {row['score']} 分")

# 寫入 CSV
with open("output.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["姓名", "年齡", "成績"])
    writer.writerow(["Alice", 20, 95])
    writer.writerow(["Bob", 22, 88])
    writer.writerows([
        ["Charlie", 21, 92],
        ["Diana", 23, 85]
    ])

# 使用 DictWriter
with open("output.csv", "w", encoding="utf-8", newline="") as f:
    fieldnames = ["姓名", "年齡", "成績"]
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerow({"姓名": "Alice", "年齡": 20, "成績": 95})
```

## JSON 檔案處理

```python
import json

data = {
    "name": "Alice",
    "age": 25,
    "skills": ["Python", "JavaScript"],
    "address": {
        "city": "台北",
        "district": "大安區"
    }
}

# 寫入 JSON
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 讀取 JSON
with open("data.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
    print(loaded["name"])
```

## 二進位檔案

```python
# 複製圖片（二進位模式）
with open("source.jpg", "rb") as src:
    with open("copy.jpg", "wb") as dst:
        # 一次讀取一塊，避免記憶體不足
        for chunk in iter(lambda: src.read(8192), b''):
            dst.write(chunk)

# 取得檔案大小
import os
file_size = os.path.getsize("example.txt")
print(f"檔案大小: {file_size} bytes")
```

## 目錄操作

```python
import os
import shutil

# 檢查路徑
print(os.path.exists("data.txt"))     # 是否存在
print(os.path.isfile("data.txt"))     # 是否為檔案
print(os.path.isdir("data"))          # 是否為目錄

# 目錄操作
os.makedirs("parent/child", exist_ok=True)  # 建立多層目錄
os.listdir(".")                              # 列出目錄內容

# 複製與移動
shutil.copy("source.txt", "dest.txt")       # 複製檔案
shutil.copytree("src_dir", "dst_dir")       # 複製整個目錄
shutil.move("source.txt", "backup/")        # 移動檔案

# 刪除
os.remove("file.txt")          # 刪除檔案
os.rmdir("empty_dir")          # 刪除空目錄
shutil.rmtree("dir_with_stuff") # 刪除目錄及其內容
```

## pathlib（現代方式）

`pathlib` 提供物件導向的路徑操作（Python 3.4+）：

```python
from pathlib import Path

# 建立路徑物件
current = Path(".")
home = Path.home()
data_file = Path("data/students.json")

# 路徑操作
path = Path("data") / "subdir" / "file.txt"
print(path.suffix)      # .txt
print(path.stem)        # file
print(path.parent)      # data/subdir

# 讀寫檔案（不用 with）
content = Path("file.txt").read_text(encoding="utf-8")
Path("output.txt").write_text("Hello", encoding="utf-8")

# 目錄操作
for file in Path(".").glob("*.py"):   # 搜尋 .py 檔案
    print(file.name)

for file in Path(".").rglob("*.txt"):  # 遞迴搜尋
    print(file)

# 建立目錄
Path("new_dir/sub_dir").mkdir(parents=True, exist_ok=True)

# 檢查
if data_file.exists() and data_file.is_file():
    print(f"檔案大小: {data_file.stat().st_size} bytes")
```

## 重點整理

| 操作 | 傳統方式 | pathlib 方式 |
|------|---------|-------------|
| 讀取檔案 | `open()` + `read()` | `Path.read_text()` |
| 寫入檔案 | `open()` + `write()` | `Path.write_text()` |
| 路徑串接 | `os.path.join()` | `/` 運算子 |
| 檢查存在 | `os.path.exists()` | `Path.exists()` |
| 列出檔案 | `os.listdir()` | `Path.glob()` |
