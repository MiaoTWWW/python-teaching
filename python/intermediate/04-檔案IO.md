# 04 - 檔案 I/O

> 🎯 **學習目標**：學習如何在 Python 中讀寫檔案，處理各種資料格式。

## 開啟與關閉檔案

### 基本檔案操作

```python
# 開啟檔案
file = open("example.txt", "r", encoding="utf-8")
content = file.read()
file.close()  # 記得關閉！
```

### 使用 with 語句（建議）

```python
# with 會自動關閉檔案
with open("example.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
# 離開 with 區塊後，檔案會自動關閉
```

## 檔案模式

| 模式 | 說明 |
|------|------|
| `"r"` | 讀取（預設）。檔案必須存在 |
| `"w"` | 寫入。會覆蓋已存在的檔案 |
| `"a"` | 附加。在檔案末尾新增內容 |
| `"x"` | 獨佔建立。檔案已存在則失敗 |
| `"b"` | 二進位模式（可與其他模式組合） |
| `"+"` | 讀寫模式（可與其他模式組合） |

```python
# 各種模式範例
with open("read.txt", "r", encoding="utf-8") as f:
    pass

with open("write.txt", "w", encoding="utf-8") as f:
    f.write("新內容")

with open("append.txt", "a", encoding="utf-8") as f:
    f.write("附加內容\n")

with open("new.txt", "x", encoding="utf-8") as f:
    f.write("獨佔建立")
```

## 讀取檔案

### 讀取方法

```python
# read() - 讀取全部內容
with open("sample.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# readline() - 逐行讀取
with open("sample.txt", "r", encoding="utf-8") as f:
    line = f.readline()
    while line:
        print(line, end="")
        line = f.readline()

# readlines() - 讀取所有行到列表
with open("sample.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    print(f"共有 {len(lines)} 行")

# 最常用的方式：直接迭代
with open("sample.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line, end="")
```

## 寫入檔案

```python
# write() - 寫入字串
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("第一行\n")
    f.write("第二行\n")

# writelines() - 寫入字串列表
lines = ["第一行\n", "第二行\n", "第三行\n"]
with open("output.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)

# print() 也可以寫入檔案
with open("output.txt", "w", encoding="utf-8") as f:
    print("這是第一行", file=f)
    print("這是第二行", file=f)
```

## CSV 檔案處理

```python
import csv

# 讀取 CSV
with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    header = next(reader)  # 跳過標題列
    for row in reader:
        name, age, score = row
        print(f"{name}: {age} 歲, 成績 {score}")

# 使用 DictReader（更方便）
with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(f"{row['name']}: {row['score']} 分")

# 寫入 CSV
with open("output.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["姓名", "年齡", "成績"])
    writer.writerow(["Alice", 20, 95])
    writer.writerow(["Bob", 22, 88])
    writer.writerows([
        ["Charlie", 21, 92],
        ["Diana", 23, 85]
    ])

# 使用 DictWriter
with open("output.csv", "w", encoding="utf-8", newline="") as f:
    fieldnames = ["姓名", "年齡", "成績"]
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerow({"姓名": "Alice", "年齡": 20, "成績": 95})
```

## JSON 檔案處理

```python
import json

data = {
    "name": "Alice",
    "age": 25,
    "skills": ["Python", "JavaScript"],
    "address": {
        "city": "台北",
        "district": "大安區"
    }
}

# 寫入 JSON
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 讀取 JSON
with open("data.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
    print(loaded["name"])
```

## 二進位檔案

```python
# 複製圖片（二進位模式）
with open("source.jpg", "rb") as src:
    with open("copy.jpg", "wb") as dst:
        # 一次讀取一塊，避免記憶體不足
        for chunk in iter(lambda: src.read(8192), b''):
            dst.write(chunk)

# 取得檔案大小
import os
file_size = os.path.getsize("example.txt")
print(f"檔案大小: {file_size} bytes")
```

## 目錄操作

```python
import os
import shutil

# 檢查路徑
print(os.path.exists("data.txt"))     # 是否存在
print(os.path.isfile("data.txt"))     # 是否為檔案
print(os.path.isdir("data"))          # 是否為目錄

# 目錄操作
os.makedirs("parent/child", exist_ok=True)  # 建立多層目錄
os.listdir(".")                              # 列出目錄內容

# 複製與移動
shutil.copy("source.txt", "dest.txt")       # 複製檔案
shutil.copytree("src_dir", "dst_dir")       # 複製整個目錄
shutil.move("source.txt", "backup/")        # 移動檔案

# 刪除
os.remove("file.txt")          # 刪除檔案
os.rmdir("empty_dir")          # 刪除空目錄
shutil.rmtree("dir_with_stuff") # 刪除目錄及其內容
```

## pathlib（現代方式）

`pathlib` 提供物件導向的路徑操作（Python 3.4+）：

```python
from pathlib import Path

# 建立路徑物件
current = Path(".")
home = Path.home()
data_file = Path("data/students.json")

# 路徑操作
path = Path("data") / "subdir" / "file.txt"
print(path.suffix)      # .txt
print(path.stem)        # file
print(path.parent)      # data/subdir

# 讀寫檔案（不用 with）
content = Path("file.txt").read_text(encoding="utf-8")
Path("output.txt").write_text("Hello", encoding="utf-8")

# 目錄操作
for file in Path(".").glob("*.py"):   # 搜尋 .py 檔案
    print(file.name)

for file in Path(".").rglob("*.txt"):  # 遞迴搜尋
    print(file)

# 建立目錄
Path("new_dir/sub_dir").mkdir(parents=True, exist_ok=True)

# 檢查
if data_file.exists() and data_file.is_file():
    print(f"檔案大小: {data_file.stat().st_size} bytes")
```

## 重點整理

| 操作 | 傳統方式 | pathlib 方式 |
|------|---------|-------------|
| 讀取檔案 | `open()` + `read()` | `Path.read_text()` |
| 寫入檔案 | `open()` + `write()` | `Path.write_text()` |
| 路徑串接 | `os.path.join()` | `/` 運算子 |
| 檢查存在 | `os.path.exists()` | `Path.exists()` |
| 列出檔案 | `os.listdir()` | `Path.glob()` |

## 🏆 經典練習題

### 檔案讀寫（open/read/write） 🟢 簡易 | 🟡 中等

1. **檔案複製**：讀取一個文字檔案的內容，寫入到另一個新檔案
   ```python
   # 實作 cp 指令的簡化版
   ```
   💡 提示：`read()` 讀取全部，`write()` 寫入

2. **逐行讀取**：讀取一個檔案，在每行前面加上行號後輸出
   💡 提示：`enumerate(file, start=1)`

3. **檔案統計**：讀取一個文字檔，統計總行數、總單字數、總字元數
   💡 提示：`len(line)`、`len(line.split())`

### with 陳述式 🟢 簡易 | 🟡 中等

4. **自動關閉**：比較使用 `with` 和不使用 `with` 的檔案操作，證明 `with` 會自動關閉檔案
   ```python
   with open("test.txt", "w") as f:
       f.write("Hello")
   # 離開 with 區塊後 f 已自動關閉
   ```
   💡 提示：檢查 `f.closed` 屬性

5. **多檔案同時操作**：用一個 `with` 同時開啟兩個檔案（讀取來源、寫入目標）
   💡 提示：`with open("src") as f1, open("dst", "w") as f2:`

### CSV 檔案處理 🟡 中等 | 🔴 困難

6. **CSV 讀取**：讀取一個 CSV 檔案，計算每一列的總和
   ```csv
   name,score1,score2,score3
   Alice,85,92,78
   Bob,90,88,95
   ```
   💡 提示：`csv.DictReader()` 或 `csv.reader()`

7. **CSV 寫入**：將一個字典列表寫入 CSV 檔案
   💡 提示：`csv.DictWriter()` 搭配 `writeheader()`

8. **CSV 過濾**：讀取一個大型 CSV，只保留符合特定條件的行（如分數 ≥ 60）
   💡 提示：逐行讀取判斷，符合條件的寫入新檔

### JSON 檔案處理 🟡 中等 | 🔴 困難

9. **JSON 讀寫**：將一個 Python 字典寫入 JSON 檔案，再讀取回來
   ```python
   data = {"name": "Alice", "scores": [85, 92, 78]}
   ```
   💡 提示：`json.dump(data, f, ensure_ascii=False, indent=2)`

10. **JSON 設定檔**：寫一支程式讀取 `config.json`，如果檔案不存在則使用預設值
    💡 提示：`try: json.load(f); except FileNotFoundError: 使用預設值`

11. **巢狀 JSON 處理**：給定一個巢狀 JSON （包含列表和子字典），寫一支函式 `get_value(data, key_path)` 用點路徑存取
    ```python
    data = {"user": {"address": {"city": "台北"}}}
    print(get_value(data, "user.address.city"))  # 台北
    ```
    💡 提示：用 `split(".")` 逐層存取

### 多檔案批次處理 🔴 困難

12. **批次重新命名**：將一個目錄中所有 `.txt` 檔案加上前綴 `backup_`
    💡 提示：`os.listdir()` + `os.rename()`

13. **日誌合併**：將多個日誌檔案按照時間戳記合併為一個檔案
    💡 提示：讀取每個檔案到列表，排序後寫出

### 目錄操作（os/pathlib） 🟡 中等

14. **目錄樹遍歷**：用 `os.walk()` 或 `Path.rglob()` 列出一個目錄下所有 Python 檔案
    💡 提示：`Path(".").rglob("*.py")`

15. **檔案大小總計**：計算一個目錄中所有 `.jpg` 檔案的總大小
    💡 提示：`Path(file).stat().st_size`

16. **磁碟使用分析**：掃描一個目錄，輸出最大檔案的前 10 名
    💡 提示：`sorted(paths, key=lambda p: p.stat().st_size, reverse=True)[:10]`
