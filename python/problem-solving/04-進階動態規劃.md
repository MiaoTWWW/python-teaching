## 1. 最大子陣列和 (Kadane's Algorithm)

給定一整數陣列，找出總和最大的**連續子陣列**：

```python
def max_subarray(nums):
    """Kadane 演算法 — O(n) 找出最大連續子陣列和"""
    max_ending_here = max_so_far = nums[0]
    for num in nums[1:]:
        max_ending_here = max(num, max_ending_here + num)
        max_so_far = max(max_so_far, max_ending_here)
    return max_so_far

print(max_subarray([-2, 1, -3, 4, -1, 2, 1, -5, 4]))  # 6（[4, -1, 2, 1]）

# 同時回傳起迄位置
def max_subarray_with_indices(nums):
    max_ending_here = max_so_far = nums[0]
    start = end = temp_start = 0
    for i in range(1, len(nums)):
        if nums[i] > max_ending_here + nums[i]:
            max_ending_here = nums[i]
            temp_start = i
        else:
            max_ending_here += nums[i]
        if max_ending_here > max_so_far:
            max_so_far = max_ending_here
            start = temp_start
            end = i
    return max_so_far, nums[start:end+1]

print(max_subarray_with_indices([-2, 1, -3, 4, -1, 2, 1, -5, 4]))
# (6, [4, -1, 2, 1])
```

## 2. 打劫問題 (House Robber)

一排連續的房子，各存有若干金錢。劫匪不能打劫相鄰的房子，求最大可打劫金額：

```python
def house_robber(nums):
    """相鄰不可搶 — O(n)"""
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    prev2, prev1 = nums[0], max(nums[0], nums[1])
    for i in range(2, len(nums)):
        curr = max(prev1, prev2 + nums[i])
        prev2, prev1 = prev1, curr
    return prev1

print(house_robber([1, 2, 3, 1]))       # 4（1+3）
print(house_robber([2, 7, 9, 3, 1]))    # 12（2+9+1）
```

### 打劫問題 II (環形排列)

房子排成一個環（第一間和最後一間相鄰）：

```python
def house_robber_ii(nums):
    """環形排列的房子，相鄰不可搶"""
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    # 要嘛不搶第一間，要嘛不搶最後一間
    return max(house_robber(nums[1:]), house_robber(nums[:-1]))

print(house_robber_ii([2, 3, 2]))       # 3（只搶第2間）
```

## 3. 戳汽球 (Burst Balloons)

有一串有權重的汽球，戳破第 i 個可得到 `nums[i-1] × nums[i] × nums[i+1]` 分數（超出範圍的視為 1）。求最大得分：

```python
def max_coins(nums):
    """戳汽球 — 區間 DP O(n³)"""
    n = len(nums)
    # 在頭尾各加一個虛擬的 1，方便處理邊界
    balloons = [1] + nums + [1]
    dp = [[0] * (n + 2) for _ in range(n + 2)]

    # 從短區間到長區間
    for length in range(1, n + 1):
        for left in range(1, n - length + 2):
            right = left + length - 1
            # 最後戳破第 k 個汽球
            for k in range(left, right + 1):
                score = (balloons[left - 1] *
                         balloons[k] *
                         balloons[right + 1])
                dp[left][right] = max(
                    dp[left][right],
                    dp[left][k - 1] + score + dp[k + 1][right]
                )

    return dp[1][n]

print(max_coins([3, 1, 5, 8]))  # 167
# 解釋：nums = [3,1,5,8] → [3,5,8] → [3,8] → [8] → []
# 分數 = 3*1*5 + 3*5*8 + 1*3*8 + 1*1*8 = 167
```

## 4. Target Sum (整數連加)

給定一串整數，可對每個數字設為正數或負數，使總和等於某個目標值，求有幾種方法：

```python
def find_target_sum_ways(nums, target):
    """整數連加 — 正負號設定方法數"""
    total = sum(nums)
    # 如果 target 超出可能範圍，直接回傳 0
    if abs(target) > total:
        return 0
    # offset 用來處理負數索引
    offset = total
    dp = [0] * (2 * total + 1)
    dp[offset] = 1  # sum=0 有 1 種方法

    for num in nums:
        next_dp = [0] * (2 * total + 1)
        for sum_val in range(-total, total + 1):
            idx = sum_val + offset
            if dp[idx]:
                next_dp[idx + num] += dp[idx]
                next_dp[idx - num] += dp[idx]
        dp = next_dp

    return dp[target + offset] if 0 <= target + offset < len(dp) else 0

print(find_target_sum_ways([1, 1, 1, 1, 1], 3))  # 5
# +1+1+1+1-1 = 3
# +1+1+1-1+1 = 3
# +1+1-1+1+1 = 3
# +1-1+1+1+1 = 3
# -1+1+1+1+1 = 3
```

## 5. 4 Keys Keyboard (產生最多個 a)

按鍵盤上的按鍵：`a`（產生一個 a）、`Ctrl+V`（貼上）、`Ctrl+A + Ctrl+C`（全選+複製，算 2 次按鍵）。求在最多 n 次按鍵下可產生的最多 a 數量：

```python
def max_a(n):
    """在 n 次按鍵內，最多可產生的 a 數量"""
    if n <= 6:
        return n

    dp = [0] * (n + 1)
    for i in range(1, 7):
        dp[i] = i  # 前 6 次直接按 a 最快

    for i in range(7, n + 1):
        # 嘗試從 j 開始全選+複製，然後貼上
        for j in range(i - 3, 0, -1):
            # 在 j 處按 Ctrl+A+C（2 次），剩下 (i-j-2) 次貼上
            dp[i] = max(dp[i], dp[j] * (i - j - 1))
            # 也可以先按 1 次 a，再 Ctrl+A+C+貼上
            # 但 dp[j]*(i-j-1) 已經涵蓋最優解
        # 也可以只是單純再按一次 a
        dp[i] = max(dp[i], dp[i - 1] + 1)

    return dp[n]

print(max_a(7))   # 9（先按 3 次 a → AAA，Ctrl+A+C+V → 6，再 Ctrl+V → 9）
print(max_a(10))  # 36（5 次 a + 全選複製 + 3 次貼上：5 × 4 = 20？不對...）
```

> 💡 關鍵觀察：最佳策略通常是在前面按幾次 a，然後交替使用「全選+複製」和「貼上」。

## 6. 刪去最少個值以成為嚴格單調遞增

給定一串整數，刪掉最少個數使其成為嚴格遞增序列（等於 n - LIS）：

```python
def min_deletions_for_strictly_increasing(nums):
    """刪除最少元素使陣列嚴格遞增 = n - LIS"""
    from bisect import bisect_left

    tails = []
    for num in nums:
        pos = bisect_left(tails, num)
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num

    return len(nums) - len(tails)

print(min_deletions_for_strictly_increasing([5, 2, 8, 6, 3, 6, 9, 7]))
# 原始長度 8，LIS = 4（2, 3, 6, 9），需刪除 4 個
```

## 7. 完全覆蓋 (Domino Tiling)

有一種 1×2 的磁磚（可以直貼或橫貼），給定某個尺寸的地板（寬 m × 高 n），有幾種完全覆蓋的貼法？

```python
def domino_tiling(m, n):
    """1×2 磁磚完全覆蓋 m×n 地板的方法數
    使用 DP + bitmask 輪廓線 DP（m 較小時適用）
    """
    if (m * n) % 2 == 1:  # 面積為奇數時不可能
        return 0

    # 確保 m 是較小的維度（用 bitmask 優化）
    if m > n:
        m, n = n, m

    # dp[current_mask] = 目前列的鋪法數
    dp = {0: 1}
    for _ in range(n):
        next_dp = {}
        for mask, count in dp.items():
            # 處理這個 mask
            def dfs(pos, next_mask, curr_mask):
                if pos >= m:
                    key = next_mask
                    next_dp[key] = next_dp.get(key, 0) + count
                    return
                if curr_mask & (1 << pos):  # 這個位置已被佔用
                    dfs(pos + 1, next_mask, curr_mask)
                else:
                    # 直放（佔用下一列的同一位置）
                    dfs(pos + 1, next_mask | (1 << pos), curr_mask)
                    # 橫放（需要 pos+1 也是空的）
                    if (pos + 1 < m and
                        not (curr_mask & (1 << (pos + 1)))):
                        dfs(pos + 2, next_mask, curr_mask)
            dfs(0, 0, mask)
        dp = next_dp

    return dp.get(0, 0)

print(domino_tiling(2, 3))   # 3
print(domino_tiling(4, 4))   # 36
```

## DP 分類速查

| 題型 | 核心策略 | 時間 |
|------|---------|------|
| **Kadane** (連續子陣列) | `dp[i] = max(num, dp[i-1] + num)` | O(n) |
| **House Robber** | `dp[i] = max(dp[i-1], dp[i-2] + nums[i])` | O(n) |
| **區間 DP** (戳汽球) | 枚舉區間內最後選的元素 | O(n³) |
| **Target Sum** | 背包思想，處理正負號 | O(n·sum) |
| **4 Keys Keyboard** | 枚舉貼上次數 | O(n²) |
