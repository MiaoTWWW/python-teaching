## 1. N 皇后問題 (N-Queens)

在 N×N 棋盤上放 N 個皇后，同一行、同一列、同一斜線上只能有一個皇后：

```python
def solve_n_queens(n):
    """N 皇后問題 — 計算所有擺法"""
    def is_safe(board, row, col):
        # 檢查同一列
        for i in range(row):
            if board[i] == col:
                return False
            # 檢查對角線
            if abs(board[i] - col) == abs(i - row):
                return False
        return True

    def backtrack(row):
        if row == n:
            result.append(board[:])
            return
        for col in range(n):
            if is_safe(board, row, col):
                board[row] = col
                backtrack(row + 1)
                board[row] = -1

    result = []
    board = [-1] * n
    backtrack(0)
    return result


def print_queens(solutions, n):
    """以棋盤格式印出解答"""
    for solution in solutions:
        for col in solution:
            line = ['.'] * n
            line[col] = 'Q'
            print(' '.join(line))
        print()

solutions = solve_n_queens(4)
print(f"共 {len(solutions)} 種擺法")
print_queens(solutions, 4)
# . Q . .
# . . . Q
# Q . . .
# . . Q .

# （第二種擺法）
# . . Q .
# Q . . .
# . . . Q
# . Q . .
```

## 2. 八皇后問題的延伸

使用 Set 最佳化檢查：

```python
def solve_n_queens_optimized(n):
    """N 皇后 — 使用 Set 加速檢查 O(n!)"""
    def backtrack(row):
        if row == n:
            result.append(board[:])
            return
        for col in range(n):
            if col in cols or (row - col) in diag1 or (row + col) in diag2:
                continue
            board.append(col)
            cols.add(col)
            diag1.add(row - col)
            diag2.add(row + col)
            backtrack(row + 1)
            cols.remove(col)
            diag1.remove(row - col)
            diag2.remove(row + col)
            board.pop()

    result = []
    board = []
    cols = set()
    diag1 = set()  # row - col（主對角線差值固定）
    diag2 = set()  # row + col（副對角線和固定）
    backtrack(0)
    return result

print(len(solve_n_queens_optimized(8)))  # 92
```

## 3. Knight's Tour（走完棋盤）

已知棋子中的「馬」只能走「日」字型，能否從某一點出發，在每個格子只走一次的情況下走遍棋盤所有格子？

```python
def knight_tour(n, start_x=0, start_y=0):
    """騎士走棋盤 — Warnsdorff 啟發式"""
    # 馬的 8 個走法
    moves = [
        (2, 1), (1, 2), (-1, 2), (-2, 1),
        (-2, -1), (-1, -2), (1, -2), (2, -1)
    ]

    board = [[-1] * n for _ in range(n)]

    def is_valid(x, y):
        return 0 <= x < n and 0 <= y < n and board[x][y] == -1

    def count_onward_moves(x, y):
        """計算從 (x,y) 還有幾個可走的下一步"""
        count = 0
        for dx, dy in moves:
            nx, ny = x + dx, y + dy
            if is_valid(nx, ny):
                count += 1
        return count

    def solve(x, y, step):
        board[x][y] = step
        if step == n * n - 1:
            return True

        # Warnsdorff 規則：優先走「下一步選擇最少」的路
        next_moves = []
        for dx, dy in moves:
            nx, ny = x + dx, y + dy
            if is_valid(nx, ny):
                degree = count_onward_moves(nx, ny)
                next_moves.append((degree, nx, ny))

        next_moves.sort()  # 優先選擇 degree 最小的

        for _, nx, ny in next_moves:
            if solve(nx, ny, step + 1):
                return True

        board[x][y] = -1  # 回溯
        return False

    if solve(start_x, start_y, 0):
        return board
    return None

def print_knight_tour(board):
    if not board:
        print("無解")
        return
    n = len(board)
    for row in board:
        print(" ".join(f"{cell:2d}" for cell in row))

# 5×5 棋盤
board = knight_tour(5)
print_knight_tour(board)
```

## 4. Word Search（拼字判斷）

給定一個字母矩陣，判斷是否可以從某個位置開始，走上下左右四個方向，連續拼出單字。每個格子只能使用一次：

```python
def exist(board, word):
    """字母矩陣中是否存在目標單字"""
    if not board or not board[0]:
        return False

    m, n = len(board), len(board[0])

    def dfs(x, y, idx):
        if idx == len(word):
            return True
        if (x < 0 or x >= m or y < 0 or y >= n or
                board[x][y] != word[idx]):
            return False

        # 標記已使用
        temp, board[x][y] = board[x][y], '#'
        found = (dfs(x + 1, y, idx + 1) or
                 dfs(x - 1, y, idx + 1) or
                 dfs(x, y + 1, idx + 1) or
                 dfs(x, y - 1, idx + 1))
        board[x][y] = temp  # 還原
        return found

    for i in range(m):
        for j in range(n):
            if board[i][j] == word[0] and dfs(i, j, 0):
                return True
    return False

board = [
    ['A', 'B', 'C', 'E'],
    ['S', 'F', 'C', 'S'],
    ['A', 'D', 'E', 'E']
]
print(exist(board, "ABCCED"))  # True
print(exist(board, "SEE"))     # True
print(exist(board, "ABCB"))    # False
```

## 5. 列出所有遞增子字串

找出一個字串（或陣列）中的所有遞增子字串：

```python
def all_increasing_substrings(s):
    """列出所有遞增子字串（連續）"""
    result = []
    n = len(s)
    for i in range(n):
        for j in range(i, n):
            # 檢查子字串 s[i:j+1] 是否遞增
            if all(s[k] < s[k + 1] for k in range(i, j)):
                result.append(s[i:j + 1])
    return result

print(all_increasing_substrings("abc"))
# ['a', 'ab', 'abc', 'b', 'bc', 'c']
```

## 6. 所有換零錢的方法

給定金額及若干種硬幣，列出所有可以換完全部金額的方法：

```python
def coin_change_all_ways(coins, amount):
    """列出所有換零錢的方法"""
    result = []

    def backtrack(remaining, start, path):
        if remaining == 0:
            result.append(path[:])
            return
        for i in range(start, len(coins)):
            if coins[i] <= remaining:
                path.append(coins[i])
                backtrack(remaining - coins[i], i, path)  # 可重複使用
                path.pop()

    backtrack(amount, 0, [])
    return result

print(coin_change_all_ways([1, 2, 5], 5))
# [[1,1,1,1,1], [1,1,1,2], [1,2,2], [5]]
```

## 7. 爬樓梯的所有走法

爬 n 階樓梯，每次可走 1 或 2 階，列出所有走法：

```python
def climb_stairs_all_ways(n):
    """爬樓梯的所有走法"""
    result = []

    def backtrack(remaining, path):
        if remaining == 0:
            result.append(path[:])
            return
        if remaining >= 1:
            path.append(1)
            backtrack(remaining - 1, path)
            path.pop()
        if remaining >= 2:
            path.append(2)
            backtrack(remaining - 2, path)
            path.pop()

    backtrack(n, [])
    return result

print(climb_stairs_all_ways(4))
# [[1,1,1,1], [1,1,2], [1,2,1], [2,1,1], [2,2]]
```

## 回溯法解題框架

```
def backtrack(候選狀態):
    if 符合終止條件:
        記錄解答
        return
    for 每個可能的選擇:
        if 選擇合法:
            做出選擇
            backtrack(更新後的狀態)
            撤銷選擇
```

### 常見應用

| 問題 | 決策樹 | 剪枝策略 |
|------|--------|---------|
| N 皇后 | 每層決定一列的位置 | 同行/同列/同斜線 |
| Knight's Tour | 每一步的 8 種走法 | Warnsdorff (最少選擇優先) |
| Word Search | 上下左右 4 方向 | 超出邊界、字元不符、已使用 |
| 排列組合 | 選或不選 / 選哪個 | 重複元素跳過 |
| 集合劃分 | 每個元素分配給哪個子集 | 總和限制、平均分配 |

## 練習

1. N 皇后：除了計算數量，還要能輸出所有棋盤圖案。
2. Knight's Tour：從棋盤的四個角出發，比較哪個角有最多解。
3. Word Search：給定一個字母矩陣和一個單字列表，找出所有存在的單字。
4. 硬幣問題：換零錢的方法中，限制最多只能用 10 個硬幣。
5. 爬樓梯：如果每次可以走 1、2 或 3 階，列出所有走法。
