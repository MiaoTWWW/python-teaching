## 1. 最短編輯距離 (Edit Distance)

兩個字串 `str1` 和 `str2`，可對 `str1` 進行三種操作：
- **增加** 1 個字元
- **刪除** 1 個字元
- **更改** 1 個字元

求最少需要幾個基本操作可將 `str1` 變成 `str2`。

```python
def edit_distance(str1, str2):
    """最短編輯距離 — DP O(m*n)"""
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    # 初始化邊界
    for i in range(m + 1):
        dp[i][0] = i  # 刪除 i 個字元
    for j in range(n + 1):
        dp[0][j] = j  # 增加 j 個字元

    # 填表
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]  # 字元相同，不需操作
            else:
                dp[i][j] = 1 + min(
                    dp[i - 1][j],      # 刪除 str1[i-1]
                    dp[i][j - 1],      # 增加 str2[j-1]
                    dp[i - 1][j - 1]   # 更改 str1[i-1] → str2[j-1]
                )
    return dp[m][n]

print(edit_distance("kitten", "sitting"))  # 3
# kitten → sitten（更改 k→s）
# sitten → sittin（更改 e→i）
# sittin → sitting（增加 g）
```

## 2. 最長迴文子序列 (LPS)

**迴文**：從左到右與從右到左內容相同。
**子序列**：內容不必連續，但順序與原字串相同。

求最長迴文子序列的長度與實際內容：

```python
def longest_palindromic_subsequence(s):
    """最長迴文子序列長度"""
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = 1  # 單一字元本身就是迴文

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            if s[i] == s[j]:
                dp[i][j] = dp[i + 1][j - 1] + 2
            else:
                dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])
    return dp[0][n - 1]


def lps_string(s):
    """回傳最長迴文子序列的實際內容"""
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = 1

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            if s[i] == s[j]:
                dp[i][j] = dp[i + 1][j - 1] + 2
            else:
                dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])

    # 回溯找出字串
    i, j = 0, n - 1
    result = []
    while i <= j:
        if s[i] == s[j]:
            result.append(s[i])
            i += 1
            j -= 1
        elif dp[i + 1][j] > dp[i][j - 1]:
            i += 1
        else:
            j -= 1
    # 前半段 + 後半段的鏡像（不含最後一個字元）
    half = len(result)
    if half == 0:
        return ""
    return "".join(result + result[-2::-1] if dp[0][n-1] % 2 == 1 else result + result[-1::-1])

# 但更簡單的做法是反過來用 LCS：
def lps_via_lcs(s):
    """LPS = LCS(s, reverse(s))"""
    from algorithms.DP.lcs import longest_common_subsequence  # 沿用 LCS
    return longest_common_subsequence(s, s[::-1])

print(longest_palindromic_subsequence("bbbab"))   # 4（bbbb 或 bbab）
```

## 3. 插入最少字元變成迴文

在字串中補上最少的字元，讓它成為迴文：

```python
def min_insertions_palindrome(s):
    """最少插入字元數使字串變成迴文"""
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            if s[i] == s[j]:
                dp[i][j] = dp[i + 1][j - 1]
            else:
                dp[i][j] = 1 + min(dp[i + 1][j], dp[i][j - 1])
    return dp[0][n - 1]

# 另一個思路：最少插入數 = n - LPS(s)
def min_insertions_via_lps(s):
    return len(s) - longest_palindromic_subsequence(s)

print(min_insertions_palindrome("mbadm"))   # 2 → "mbdadbm"
```

## 4. 字串樣式比對 (Wildcard Matching)

`*` 代表 0 個或多個任意字元，`?` 代表 1 個任意字元：

```python
def is_match(text, pattern):
    """Wildcard 比對，? 代表 1 個任意字元，* 代表任意長度"""
    m, n = len(text), len(pattern)
    dp = [[False] * (n + 1) for _ in range(m + 1)]
    dp[0][0] = True

    # 初始化：pattern 開頭的 * 可以對應空字串
    for j in range(1, n + 1):
        if pattern[j - 1] == '*':
            dp[0][j] = dp[0][j - 1]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if pattern[j - 1] == '*':
                # * 可以匹配 0 個（dp[i][j-1]）或多個（dp[i-1][j]）
                dp[i][j] = dp[i][j - 1] or dp[i - 1][j]
            elif pattern[j - 1] == '?' or text[i - 1] == pattern[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]

    return dp[m][n]

print(is_match("aa", "a"))       # False
print(is_match("aa", "*"))       # True
print(is_match("cb", "?a"))      # False
print(is_match("adceb", "*a*b")) # True
```

## 5. 區間標籤切割 (Partition Labels)

給定一個由小寫字母組成的字串，將它切割成「儘可能多的片段」，滿足**每個字母最多只能出現在一個片段中**：

```python
def partition_labels(s):
    """區間標籤切割 — 回傳每個片段的長度"""
    # 先記錄每個字母最後出現的位置
    last_pos = {}
    for i, ch in enumerate(s):
        last_pos[ch] = i

    result = []
    start = end = 0
    for i, ch in enumerate(s):
        end = max(end, last_pos[ch])  # 擴展當前片段結尾
        if i == end:  # 當前片段結束
            result.append(end - start + 1)
            start = i + 1

    return result

print(partition_labels("ababcbacadefegdehijhklij"))
# [9, 7, 8]
# "ababcbaca"、"defegde"、"hijhklij"
```

## 6. 所有合法括號組合 (Generate Parentheses)

輸入 n 對括號，輸出所有合法的括號組合：

```python
def generate_parentheses(n):
    """產生 n 對括號的所有合法組合"""
    result = []

    def backtrack(s, left, right):
        if len(s) == 2 * n:
            result.append(s)
            return
        if left < n:
            backtrack(s + '(', left + 1, right)
        if right < left:
            backtrack(s + ')', left, right + 1)

    backtrack('', 0, 0)
    return result

print(generate_parentheses(3))
# ['((()))', '(()())', '(())()', '()(())', '()()()']
```

## 7. 字元間插入星號

給定一個字串，在兩兩字元之間可以選擇插入或不插入星號，列出所有可能組合：

```python
def insert_stars(s):
    """字元間插入星號的所有可能"""
    if len(s) <= 1:
        return [s]

    result = []
    def backtrack(i, current):
        if i == len(s) - 1:
            result.append(current + s[i])
            return
        # 不插入星號
        backtrack(i + 1, current + s[i])
        # 插入星號
        backtrack(i + 1, current + s[i] + '*')

    backtrack(0, "")
    return result

print(insert_stars("ab"))
# ['ab', 'a*b']
print(insert_stars("abc"))
# ['abc', 'ab*c', 'a*bc', 'a*b*c']
```

## 8. 相鄰重複字元處理

### 相同字元只保留第一個

```python
def keep_first_duplicate(s):
    """連續相同字元只保留第一個"""
    if not s:
        return ""
    result = [s[0]]
    for ch in s[1:]:
        if ch != result[-1]:
            result.append(ch)
    return "".join(result)

print(keep_first_duplicate("aaabbbccc"))  # "abc"
```

### 相同字元只保留最後一個

```python
def keep_last_duplicate(s):
    """連續相同字元只保留最後一個"""
    if not s:
        return ""
    result = []
    i = 0
    while i < len(s):
        # 找到連續相同字元的最後一個
        j = i
        while j < len(s) and s[j] == s[i]:
            j += 1
        result.append(s[j - 1])  # 取最後一個
        i = j
    return "".join(result)

print(keep_last_duplicate("aaabbbccc"))  # "abc"
```

### Run-Length 壓縮

```python
def rle_encode(s):
    """Run-Length Encoding：將 aaabbc 壓縮為 a3b2c1"""
    if not s:
        return ""
    result = []
    count = 1
    for i in range(1, len(s)):
        if s[i] == s[i - 1]:
            count += 1
        else:
            result.append(f"{s[i-1]}{count}")
            count = 1
    result.append(f"{s[-1]}{count}")
    return "".join(result)

print(rle_encode("aaabbc"))  # "a3b2c1"
```
