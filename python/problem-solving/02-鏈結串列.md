## 什麼是鏈結串列？

鏈結串列是由**節點 (Node)** 串接而成的線性資料結構。每個節點包含：
- **資料 (data / val)**
- **指向下一個節點的指標 (next)**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

### 鏈結串列 vs 陣列

| 操作 | 陣列 (List) | 鏈結串列 |
|------|------------|----------|
| 隨機存取 | O(1) | O(n) |
| 開頭插入 | O(n) | O(1) |
| 中間插入 | O(n) | O(1) (已知位置) |
| 搜尋 | O(n) | O(n) |
| 記憶體 | 連續區塊 | 分散區塊 |

## 建立與走訪鏈結串列

### 建立一串列

```python
# 手動建立：1 → 2 → 3 → None
head = ListNode(1)
head.next = ListNode(2)
head.next.next = ListNode(3)
```

### 從串列建立

```python
def create_linked_list(arr):
    """從 Python 串列建立鏈結串列，回傳 head"""
    if not arr:
        return None
    head = ListNode(arr[0])
    current = head
    for val in arr[1:]:
        current.next = ListNode(val)
        current = current.next
    return head

head = create_linked_list([1, 2, 3, 4, 5])
```

### 走訪與列印

```python
def traverse(head):
    """走訪並印出所有節點值"""
    current = head
    while current:
        print(current.val, end=" → ")
        current = current.next
    print("None")

# 輸出：1 → 2 → 3 → 4 → 5 → None
traverse(head)
```

### 計算節點數量

```python
def count_nodes(head):
    """計算鏈結串列的節點數"""
    count = 0
    current = head
    while current:
        count += 1
        current = current.next
    return count

print(count_nodes(head))  # 5
```

## 基本操作

### 在開頭插入

```python
def insert_at_head(head, val):
    """在鏈結串列開頭插入新節點"""
    new_node = ListNode(val)
    new_node.next = head
    return new_node  # 新的 head

head = insert_at_head(head, 0)
traverse(head)  # 0 → 1 → 2 → 3 → 4 → 5 → None
```

### 在結尾插入

```python
def insert_at_tail(head, val):
    """在鏈結串列結尾插入新節點"""
    if not head:
        return ListNode(val)
    current = head
    while current.next:
        current = current.next
    current.next = ListNode(val)
    return head

head = insert_at_tail(head, 6)
traverse(head)  # 0 → 1 → 2 → 3 → 4 → 5 → 6 → None
```

### 刪除指定值的節點

```python
def delete_node(head, val):
    """刪除第一個值等於 val 的節點"""
    if not head:
        return None
    if head.val == val:
        return head.next
    current = head
    while current.next:
        if current.next.val == val:
            current.next = current.next.next
            return head
        current = current.next
    return head

head = delete_node(head, 3)
traverse(head)  # 0 → 1 → 2 → 4 → 5 → 6 → None
```

## 經典題型

### 反轉鏈結串列

```python
def reverse_list(head):
    """反轉鏈結串列"""
    prev = None
    current = head
    while current:
        next_temp = current.next  # 暫存下一個節點
        current.next = prev       # 反轉指標方向
        prev = current            # prev 往前移動
        current = next_temp       # current 往前移動
    return prev  # prev 變成新的 head

head = reverse_list(create_linked_list([1, 2, 3, 4]))
traverse(head)  # 4 → 3 → 2 → 1 → None
```

### 找出中間節點（快慢指標）

```python
def middle_node(head):
    """使用快慢指標找出中間節點"""
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow  # 偶數長度時回傳第二個中間節點

head = create_linked_list([1, 2, 3, 4, 5])
print(middle_node(head).val)  # 3
```

### 偵測是否有環

```python
def has_cycle(head):
    """Floyd 環檢測演算法（快慢指標）"""
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    return False
```

### 合併兩個排序串列

```python
def merge_two_lists(l1, l2):
    """合併兩個遞增排序的鏈結串列"""
    dummy = ListNode(0)
    current = dummy
    while l1 and l2:
        if l1.val <= l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    current.next = l1 if l1 else l2
    return dummy.next

l1 = create_linked_list([1, 3, 5])
l2 = create_linked_list([2, 4, 6])
merged = merge_two_lists(l1, l2)
traverse(merged)  # 1 → 2 → 3 → 4 → 5 → 6 → None
```

### 刪倒數第 N 個節點

```python
def remove_nth_from_end(head, n):
    """刪除倒數第 n 個節點（一次走訪）"""
    dummy = ListNode(0)
    dummy.next = head
    fast = slow = dummy
    # fast 先走 n+1 步
    for _ in range(n + 1):
        fast = fast.next
    # 一起走直到 fast 到結尾
    while fast:
        fast = fast.next
        slow = slow.next
    # slow 的下一個就是要刪除的節點
    slow.next = slow.next.next
    return dummy.next
```

## 雙向鏈結串列

```python
class DoublyListNode:
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next

def array_to_doubly(arr):
    """從串列建立雙向鏈結串列"""
    if not arr:
        return None
    head = DoublyListNode(arr[0])
    current = head
    for val in arr[1:]:
        new_node = DoublyListNode(val)
        current.next = new_node
        new_node.prev = current
        current = new_node
    return head

def traverse_doubly(head):
    """走訪雙向鏈結串列（雙向）"""
    print("正向：", end="")
    current = head
    while current:
        print(current.val, end=" ⇄ ")
        current = current.next
    print("None")
```

## 時間複雜度

| 操作 | 單向 | 雙向 |
|------|------|------|
| 存取頭節點 | O(1) | O(1) |
| 存取尾節點 | O(n) | O(1) |
| 插入開頭 | O(1) | O(1) |
| 插入結尾 | O(n) | O(1) |
| 刪除節點 | O(n) | O(1)（已知節點） |
| 搜尋 | O(n) | O(n) |
