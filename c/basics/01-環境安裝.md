# 01 - 環境安裝與設定

> 🎯 **學習目標**：在本章中，你將學會安裝 C 語言編譯器、設定開發環境，並成功編譯執行你的第一個 C 程式。

---

## 什麼是 C 語言？

C 語言由 **Dennis Ritchie** 於 1972 年在貝爾實驗室開發，最初是為了重新實作 UNIX 作業系統而設計。時至今日，C 語言仍然是：

- **作業系統核心**的首選語言（Linux、Windows 核心）
- **嵌入式系統**與韌體開發的主流工具
- **編譯器與直譯器**的實作語言
- **資料庫引擎**（如 SQLite）的核心語言

> 💡 **提示**：學好 C 語言，你將更深入地理解電腦如何運作——記憶體管理、資料在底層的表示方式，這些知識對任何程式語言都有幫助。

## 安裝 C 編譯器

C 語言是**編譯式語言**，你需要一個編譯器將原始碼轉換成可執行的機器碼。最常用的 C 編譯器是 **GCC（GNU Compiler Collection）**。

### Windows 系統：安裝 MinGW-w64

MinGW（Minimalist GNU for Windows）是 Windows 上的 GCC 移植版本。

**方法一：使用 MSYS2（推薦）**

1. 前往 [MSYS2 官網](https://www.msys2.org/) 下載安裝檔
2. 依照安裝程式指示完成安裝
3. 開啟 **MSYS2 UCRT64** 終端機，執行：
   ```bash
   pacman -S mingw-w64-ucrt-x86_64-gcc
   ```
4. 將 `C:\msys64\ucrt64\bin` 加入系統 PATH 環境變數
5. 驗證安裝：
   ```bash
   gcc --version
   ```

**方法二：使用 TDM-GCC（簡單快速）**

1. 前往 [TDM-GCC 官網](https://jmeubank.github.io/tdm-gcc/) 下載安裝檔
2. 執行安裝程式，全部使用預設選項
3. 驗證安裝：
   ```bash
   gcc --version
   ```

### macOS 系統

1. 打開終端機，安裝 Xcode Command Line Tools：
   ```bash
   xcode-select --install
   ```
2. 這會安裝 clang（與 GCC 相容的編譯器）
3. 驗證安裝：
   ```bash
   gcc --version
   ```

### Linux 系統

```bash
# Debian / Ubuntu
sudo apt update
sudo apt install gcc

# Fedora / RHEL
sudo dnf install gcc

# Arch Linux
sudo pacman -S gcc
```

## 開發環境設定（VS Code）

雖然你可以用任何文字編輯器寫 C，但 **VS Code** 提供了最友善的開發體驗。

### 安裝 VS Code

1. 前往 [code.visualstudio.com](https://code.visualstudio.com/) 下載安裝
2. 安裝完成後，開啟 VS Code

### 安裝必要擴充功能

在 VS Code 中按下 `Ctrl+Shift+X` 開啟擴充功能面板，安裝以下套件：

| 擴充功能 | 用途 |
|---------|------|
| **C/C++**（Microsoft） | 語法高亮、IntelliSense、除錯支援 |
| **Code Runner** | 一鍵編譯執行 |
| **C/C++ Extension Pack** | 額外的 C/C++ 工具集 |

### 設定編譯任務（tasks.json）

建立一個 `.vscode` 資料夾在你的專案目錄中，新增 `tasks.json`：

```json
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "編譯 C 程式",
      "type": "shell",
      "command": "gcc",
      "args": [
        "-g",
        "${file}",
        "-o",
        "${fileDirname}\\${fileBasenameNoExtension}.exe"
      ],
      "group": {
        "kind": "build",
        "isDefault": true
      }
    }
  ]
}
```

之後按 `Ctrl+Shift+B` 即可編譯目前的 C 檔案。

## 第一個 C 程式：Hello, World!

建立一個新檔案 `hello.c`，輸入以下內容：

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

### 程式碼解說

| 部分 | 說明 |
|------|------|
| `#include <stdio.h>` | 引入**標準輸入輸出函式庫**，讓我們可以使用 `printf` |
| `int main()` | **主函式**——程式執行的起點，每個 C 程式都需要一個 main |
| `{ }` | 大括號包住的範圍是函式的**本體** |
| `printf("Hello, World!\n");` | 在螢幕上印出文字，`\n` 代表**換行** |
| `return 0;` | 回傳 0 給作業系統，表示程式**正常結束** |

> ⚠️ **注意**：C 語言的每一行敘述**必須以分號 `;` 結尾**！這是初學者最常犯的錯誤。

## 編譯與執行

### 使用命令列編譯

開啟終端機（命令提示字元 / PowerShell / bash），切換到 `hello.c` 所在的目錄：

```bash
# 編譯：將 hello.c 轉換成可執行檔
gcc hello.c -o hello

# 執行（Windows）
hello.exe

# 執行（macOS / Linux）
./hello
```

你會看到輸出：

```
Hello, World!
```

### gcc 常用參數

| 參數 | 用途 |
|------|------|
| `-o 檔名` | 指定輸出檔名（預設是 `a.exe`） |
| `-Wall` | 開啟所有常見警告 |
| `-Wextra` | 開啟額外警告 |
| `-g` | 產生除錯資訊（供 GDB 使用） |
| `-O2` | 最佳化等級 2（提升執行效能） |

範例：

```bash
gcc hello.c -o hello -Wall -Wextra
```

### 使用 Code Runner 執行

在 VS Code 中安裝 Code Runner 後：

1. 開啟 `hello.c`
2. 按 `Ctrl+Alt+N`（或右上角的 ▶ 按鈕）
3. 輸出會直接顯示在 OUTPUT 面板中

## 常見錯誤排除

### ❌ 錯誤：缺少分號

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n")  // 忘記加 ;
    return 0;
}
```

**錯誤訊息**：`error: expected ';' before 'return'`

**解決方法**：每行敘述結尾加上 `;`。

### ❌ 錯誤：找不到 stdio.h

```c
#include <stdio.h>   // 打錯字
```

**錯誤訊息**：`fatal error: stdio.h: No such file or directory`

**解決方法**：檢查拼寫是否正確，正確定義是 `<stdio.h>`。

### ❌ 錯誤：main 回傳型別錯誤

```c
main() {              // 省略 int（舊式寫法，不建議）
    printf("Hello");
}
```

現代 C 標準要求明確寫出回傳型別：

```c
int main() {
    printf("Hello");
    return 0;
}
```

### ❌ 錯誤：中文字元問題

```c
printf("哈囉，世界！\n");  // 某些終端機可能無法正確顯示中文
```

如果輸出亂碼，確保原始檔儲存為 UTF-8 編碼，並在終端機中使用支援 UTF-8 的字型。

## 本章重點整理

| 概念 | 重點 |
|------|------|
| 編譯器 | 將 C 原始碼轉換成可執行檔的工具 |
| GCC | 最常用的 C 編譯器 |
| `#include` | 引入函式庫的指令 |
| `main()` | 程式的進入點 |
| `printf()` | 輸出文字到畫面的函式 |
| `return 0` | 表示程式正常結束 |
| 分號 `;` | 每一行敘述的結束符號 |

## 練習題

### 練習 1：修改輸出內容

修改 `hello.c`，讓它輸出你的姓名，例如：

```
你好，我是小明！
```

### 練習 2：加入多行輸出

撰寫一個程式，輸出以下三行文字：

```
C 語言學習開始！
加油！
我一定可以的！
```

**提示**：使用多個 `printf`，或在字串中使用 `\n`。

### 練習 3：計算機練習

撰寫一個程式，計算 `5 + 3`、`10 - 4`、`6 * 7`、`20 / 4` 的結果並輸出。例如：

```
5 + 3 = 8
10 - 4 = 6
6 * 7 = 42
20 / 4 = 5
```

**提示**：`printf("%d", 5 + 3);` 可以印出運算結果。
