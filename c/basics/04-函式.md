# 04 - 函式

> 🎯 **學習目標**：學會定義與使用函式，理解傳值呼叫與變數作用域，建立模組化程式設計的基礎。

## 什麼是函式？

函式（Function）是一段**可重複使用的程式碼區塊**，接受輸入（參數），執行特定工作，並回傳結果。

使用函式的好處：

| 優點 | 說明 |
|------|------|
| 🔄 **重複使用** | 相同的程式碼不需重複寫 |
| 📦 **模組化** | 將複雜問題拆解成小單元 |
| 📖 **可讀性** | 函式名稱說明其功能 |
| 🔧 **維護性** | 修改函式內部不影響外部 |

### C 語言的函式結構

```c
#include <stdio.h>

// 函式定義
// 回傳型別  函式名稱(參數列表) {
//     函式本體
//     return 回傳值;
// }

// 範例：計算兩個整數的和
int add(int a, int b) {
    int sum = a + b;
    return sum;
}

int main() {
    int result = add(3, 5);      // 呼叫函式
    printf("3 + 5 = %d\n", result);
    return 0;
}
```

## 函式定義與呼叫

### 基本語法

```c
回傳型別 函式名稱(參數型別 參數名稱1, 參數型別 參數名稱2, ...) {
    // 函式本體
    return 回傳值;  // 如果回傳型別不是 void
}
```

### 完整範例

```c
#include <stdio.h>

// 有參數、有回傳值
int square(int n) {
    return n * n;
}

// 有參數、無回傳值
void print_hello(int times) {
    for (int i = 0; i < times; i++) {
        printf("Hello! ");
    }
    printf("\n");
}

// 無參數、有回傳值
int get_current_year() {
    return 2026;
}

// 無參數、無回傳值
void show_banner() {
    printf("==================\n");
    printf("   C 語言學習系統  \n");
    printf("==================\n");
}

int main() {
    show_banner();

    int x = 5;
    printf("%d 的平方 = %d\n", x, square(x));

    print_hello(3);

    int year = get_current_year();
    printf("今年是 %d 年\n", year);

    return 0;
}
```

## 函式原型（Prototype）

在 C 語言中，函式在呼叫前必須**先宣告**。有三種方式：

### 方式一：函式定義放在呼叫之前（前面範例就是用這種）

但如果函式很多，main 就會被擠到最下面。

### 方式二：使用函式原型宣告（推薦）

```c
#include <stdio.h>

// 函式原型（宣告）：告訴編譯器這個函式的存在
int add(int a, int b);
void print_result(int result);

int main() {
    int sum = add(10, 20);
    print_result(sum);
    return 0;
}

// 函式定義可以放在 main 之後
int add(int a, int b) {
    return a + b;
}

void print_result(int result) {
    printf("結果：%d\n", result);
}
```

### 方式三：函式原型中可省略參數名稱

```c
int add(int, int);     // 編譯器只需要知道型別
void print_result(int);
```

> 💡 **提示**：在大型程式中，通常將函式原型放在**標頭檔（`.h`）**中，函式實作放在**原始檔（`.c`）**中。但在本課程中，我們先將所有程式碼寫在一個檔案中。

## return 陳述

`return` 用來**結束函式**並**回傳值**：

```c
#include <stdio.h>

// 回傳整數
int max(int a, int b) {
    if (a > b) {
        return a;   // 提前回傳
    }
    return b;
}

// void 函式可以省略 return，或使用 return; 提前結束
void print_status(int score) {
    if (score < 0 || score > 100) {
        printf("分數無效！\n");
        return;         // 提前結束函式
    }
    printf("分數：%d\n", score);
    // 結尾可以省略 return
}

int main() {
    printf("最大值：%d\n", max(10, 20));

    print_status(-5);
    print_status(85);

    return 0;
}
```

> ⚠️ **注意**：回傳型別為 `void` 的函式也可以使用 `return;`（不帶值）來提前結束。非 `void` 函式**必須**回傳對應型別的值。

## 傳值呼叫（Call by Value）

C 語言中，函式參數預設是**傳值呼叫**——傳入的是**值的副本**，函式內部的修改不影響原變數：

```c
#include <stdio.h>

void try_to_change(int x) {
    x = 999;  // 只修改了副本
    printf("函式內部：x = %d\n", x);
}

int main() {
    int num = 10;
    printf("呼叫前：num = %d\n", num);

    try_to_change(num);

    printf("呼叫後：num = %d\n", num);  // num 仍然是 10！

    return 0;
}
```

### 記憶體示意圖

```
呼叫前：
  main 中的 num → [  10  ]

呼叫 try_to_change(num)：
  ┌─────────────────────────┐
  │ try_to_change 的 x → [ 10 ] ← 複製一份  │
  └─────────────────────────┘
  x = 999 之後：
  ┌─────────────────────────┐
  │ try_to_change 的 x → [999]               │
  └─────────────────────────┘
  main 中的 num → [  10  ] ← 不受影響
```

> 💡 **提示**：如果想讓函式修改原變數，需要用到**指標**（第六章會詳細介紹）。

## 變數作用域（Scope）

### 區域變數（Local Variable）

在函式或區塊內部宣告的變數：

```c
#include <stdio.h>

void func() {
    int local = 10;     // 區域變數：只在 func() 內有效
    printf("%d\n", local);
}

int main() {
    // printf("%d\n", local);  // ❌ 錯誤：local 未定義
    func();
    return 0;
}
```

### 區塊作用域（Block Scope）

```c
#include <stdio.h>

int main() {
    int x = 10;

    if (x > 5) {
        int y = 20;         // y 只在這個大括號內有效
        printf("x = %d, y = %d\n", x, y);
    }

    // printf("%d\n", y);   // ❌ 錯誤：y 未定義

    for (int i = 0; i < 3; i++) {  // C99 之後，i 只在 for 迴圈內有效
        printf("%d ", i);
    }
    // printf("%d\n", i);   // ❌ 錯誤：i 未定義

    return 0;
}
```

### 全域變數（Global Variable）

在所有函式**外部**宣告的變數，整個程式都可存取：

```c
#include <stdio.h>

int global_count = 0;   // 全域變數

void increment() {
    global_count++;      // 可以存取全域變數
}

int main() {
    increment();
    increment();
    increment();
    printf("計數：%d\n", global_count);  // 輸出 3
    return 0;
}
```

> ⚠️ **注意**：全域變數雖然方便，但**過度使用會讓程式難以維護**。建議只在真正需要的地方使用，並盡量以參數傳遞代替。

### static 變數

`static` 有兩種用法：

1. **靜態區域變數**：變數的生命週期延長到整個程式，但作用域不變
2. **靜態全域變數/函式**：限制只能在同一個檔案中使用

```c
#include <stdio.h>

void counter() {
    static int count = 0;   // static：只初始化一次
    count++;
    printf("第 %d 次呼叫\n", count);
}

int main() {
    counter();  // 第 1 次呼叫
    counter();  // 第 2 次呼叫
    counter();  // 第 3 次呼叫
    return 0;
}
```

```
static int count 的記憶體行為：
第一次呼叫 counter()  → count = 0 → 1
第二次呼叫 counter()  → count = 1 → 2  ← 值被保留！
第三次呼叫 counter()  → count = 2 → 3
```

### 變數遮蔽（Variable Shadowing）

當區域變數與全域變數同名時，區域變數會**遮蔽**全域變數：

```c
#include <stdio.h>

int x = 100;  // 全域變數

int main() {
    int x = 200;  // 區域變數，遮蔽了全域的 x
    printf("區域 x = %d\n", x);

    // 使用 :: 或 extern 可以存取全域變數（但 C 沒有 :: 運算子）
    // 可以用 { } 區塊來控制作用域
    {
        extern int x;  // 宣告使用全域 x
        printf("全域 x = %d\n", x);  // 100
    }

    return 0;
}
```

## 遞迴函式（Recursion）

遞迴是指函式**呼叫自己**。這是一種強大的解決問題方式，特別適合處理具有**重複結構**的問題。

### 階乘（Factorial）

```c
#include <stdio.h>

// n! = n × (n-1) × ... × 2 × 1
// 遞迴定義：n! = n × (n-1)!，且 0! = 1

int factorial(int n) {
    // 基本情況（終止條件）
    if (n <= 1) {
        return 1;
    }
    // 遞迴情況
    return n * factorial(n - 1);
}

int main() {
    for (int i = 0; i <= 10; i++) {
        printf("%d! = %d\n", i, factorial(i));
    }
    return 0;
}
```

### 遞迴呼叫的過程（factorial(4)）

```
factorial(4)
  → 4 * factorial(3)
    → 3 * factorial(2)
      → 2 * factorial(1)
        → 1           ← 終止條件
      → 2 * 1 = 2
    → 3 * 2 = 6
  → 4 * 6 = 24        ← 回傳結果
```

### 費氏數列（Fibonacci）

```c
#include <stdio.h>

// 費氏數列：1, 1, 2, 3, 5, 8, 13, 21, ...
// f(1) = 1, f(2) = 1
// f(n) = f(n-1) + f(n-2)

int fib(int n) {
    if (n <= 2) {
        return 1;
    }
    return fib(n - 1) + fib(n - 2);
}

int main() {
    printf("費氏數列前 10 項：\n");
    for (int i = 1; i <= 10; i++) {
        printf("f(%d) = %d\n", i, fib(i));
    }
    return 0;
}
```

> ⚠️ **注意**：遞迴雖然寫法優雅，但**效率可能較差**（如費氏數列有大量重複計算），且會消耗較多堆疊記憶體。實戰中，費氏數列通常用迴圈實作。

### 遞迴 vs 迴圈

| 比較 | 遞迴 | 迴圈 |
|------|------|------|
| 程式碼 | 簡潔、數學定義直觀 | 需要較多變數管理 |
| 效能 | 函式呼叫有額外開銷 | 較快 |
| 記憶體 | 每層遞迴消耗堆疊空間 | 固定記憶體 |
| 適用場景 | 樹、圖形、分治法 | 線性重複操作 |

## 本章重點整理

| 概念 | 重點 |
|------|------|
| 函式定義 | `回傳型別 名稱(參數) { 本體 }` |
| 函式原型 | 在呼叫前先宣告函式 |
| `return` | 回傳值並結束函式 |
| 傳值呼叫 | 參數是副本，函式內不影響原變數 |
| 區域變數 | 在函式或區塊內有效 |
| 全域變數 | 在檔案中任何地方都可存取 |
| `static` | 延長區域變數生命週期，或限制檔案範圍 |
| 遞迴 | 函式呼叫自己，需有終止條件 |
