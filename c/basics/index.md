## 學習建議

- **循序漸進**：請依照章節順序學習，每一章都建立在前一章的基礎上
- **動手實作**：每個程式碼範例都請親自輸入並編譯執行
- **善用筆記**：C 語言的細節較多，建議邊學邊做筆記
- **錯誤是老師**：編譯錯誤訊息是學習 C 語言最好的教材

## 必備工具

要開始寫 C 語言，你只需要：

1. **一個文字編輯器**（VS Code、Vim、Sublime Text...）
2. **一個 C 編譯器**（GCC、MinGW、Clang...）
3. **一份 terminal**（命令提示字元、PowerShell、bash...）
