# 13 - 鏈結串列實作

> 🎯 **學習目標**：理解鏈結串列（Linked List）的資料結構與操作，學會使用動態記憶體配置建立靈活的資料結構，並實作完整的增刪改查功能。

---

## 為什麼需要鏈結串列？

到目前為止，我們學過的陣列（Array）有一個主要的限制：

```c
int arr[5] = {1, 2, 3, 4, 5};
// ❌ 陣列大小固定，無法動態增減
// ❌ 在中間插入元素需要搬移大量資料
// ❌ 在中間刪除元素也需要搬移大量資料
```

**鏈結串列**解決了這些問題：

| 比較 | 陣列 | 鏈結串列 |
|------|------|---------|
| 大小 | 固定（編譯時決定） | 動態（執行時可增減） |
| 記憶體 | 連續區塊 | 分散區塊（透過指標鏈結） |
| 隨機存取 | ✅ `arr[5]` O(1) | ❌ 需要從頭走訪 O(n) |
| 插入/刪除（中間） | ❌ O(n) | ✅ O(1)（已知位置） |
| 記憶體效率 | ✅ 無額外開銷 | ❌ 每個節點多存一個指標 |

> 💡 **一句話總結**：陣列擅長「隨機存取」，鏈結串列擅長「頻繁插入刪除」。

## 單向鏈結串列（Singly Linked List）

### 基本概念

每個節點（Node）包含兩部分：
1. **資料**：儲存實際的資料
2. **指標**：指向下一個節點

```
┌──────┬──────┐    ┌──────┬──────┐    ┌──────┬──────┐
│ 4    │  ●──────→ │ 7    │  ●──────→ │ 3    │ NULL │
└──────┴──────┘    └──────┴──────┘    └──────┴──────┘
  head                next                tail
```

### 定義節點結構

```c
// 節點結構：一個 int 資料 + 一個指向下一個節點的指標
typedef struct Node {
    int data;           // 資料欄位
    struct Node *next;  // 指向下一個節點的指標
} Node;

// 我們也可以把串列本身包成一個結構（管理 head）
typedef struct LinkedList {
    Node *head;         // 指向第一個節點
    int size;           // 節點個數（方便查詢長度）
} LinkedList;
```

> 💡 **思考**：為什麼 `next` 是 `struct Node*` 而不是 `Node*`？因為在 `typedef` 還沒完成之前，編譯器還不知道 `Node` 是什麼，所以只能用完整的 `struct Node` 來宣告。

## 基本操作實作

### 1️⃣ 初始化

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;

typedef struct {
    Node *head;
    int size;
} LinkedList;

// 初始化空的鏈結串列
void init_list(LinkedList *list) {
    list->head = NULL;
    list->size = 0;
}
```

### 2️⃣ 建立新節點（輔助函式）

```c
// 建立一個新節點（回傳節點指標）
Node* create_node(int data) {
    Node *new_node = (Node*)malloc(sizeof(Node));
    if (new_node == NULL) {
        printf("記憶體配置失敗！\n");
        exit(1);
    }
    new_node->data = data;
    new_node->next = NULL;
    return new_node;
}
```

### 3️⃣ 插入節點

#### 在頭部插入（Push Front）

```
步驟：
1. 建立新節點
2. 新節點的 next 指向目前的 head
3. 更新 head 為新節點

[插入前]
  head → [7] → [3] → NULL

[步驟 1-2]
  [4] → head → [7] → [3] → NULL

[步驟 3]
  head → [4] → [7] → [3] → NULL
```

```c
void push_front(LinkedList *list, int data) {
    Node *new_node = create_node(data);
    new_node->next = list->head;  // 新節點指向舊的 head
    list->head = new_node;        // head 指向新節點
    list->size++;
}
```

#### 在尾部插入（Push Back）

```
head → [4] → [7] → [3] → NULL
                          ↑
                          tail（要找到最後一個節點）
```

```c
void push_back(LinkedList *list, int data) {
    Node *new_node = create_node(data);

    // 如果是空的串列，直接設為 head
    if (list->head == NULL) {
        list->head = new_node;
    } else {
        // 走訪到最後一個節點
        Node *current = list->head;
        while (current->next != NULL) {
            current = current->next;
        }
        // current 現在是最後一個節點
        current->next = new_node;
    }
    list->size++;
}
```

#### 在指定位置插入

```c
void insert_at(LinkedList *list, int index, int data) {
    if (index < 0 || index > list->size) {
        printf("索引超出範圍！\n");
        return;
    }

    if (index == 0) {
        push_front(list, data);
        return;
    }

    Node *new_node = create_node(data);
    Node *current = list->head;

    // 走訪到 index-1 的位置（前一個節點）
    for (int i = 0; i < index - 1; i++) {
        current = current->next;
    }

    // 在 current 之後插入新節點
    new_node->next = current->next;
    current->next = new_node;
    list->size++;
}
```

> 💡 **記憶口訣**：插入的關鍵是「先接再斷」——新節點先指向後面的節點（先接上），前一個節點再指向新節點（再斷開舊鏈結）。

### 4️⃣ 刪除節點

#### 刪除頭部節點

```c
void pop_front(LinkedList *list) {
    if (list->head == NULL) {
        printf("串列為空，無法刪除！\n");
        return;
    }

    Node *temp = list->head;       // 記住要被刪除的節點
    list->head = list->head->next; // head 指向下一個節點
    free(temp);                    // 釋放記憶體
    list->size--;
}
```

#### 刪除指定位置的節點

```c
void delete_at(LinkedList *list, int index) {
    if (list->head == NULL) {
        printf("串列為空，無法刪除！\n");
        return;
    }

    if (index < 0 || index >= list->size) {
        printf("索引超出範圍！\n");
        return;
    }

    if (index == 0) {
        pop_front(list);
        return;
    }

    Node *current = list->head;
    for (int i = 0; i < index - 1; i++) {
        current = current->next;
    }

    // current 是要刪除節點的前一個節點
    Node *to_delete = current->next;       // 要刪除的節點
    current->next = to_delete->next;       // 跳過要刪除的節點
    free(to_delete);                       // 釋放記憶體
    list->size--;
}
```

> ⚠️ **注意**：刪除節點後**一定要 free**，否則會造成記憶體洩漏！

### 5️⃣ 查詢與走訪

```c
// 走訪並印出所有節點
void print_list(LinkedList *list) {
    Node *current = list->head;

    printf("串列內容 [大小=%d]: ", list->size);
    while (current != NULL) {
        printf("%d", current->data);
        if (current->next != NULL) {
            printf(" → ");
        }
        current = current->next;
    }
    printf("\n");
}

// 根據索引取得資料
int get_at(LinkedList *list, int index) {
    if (index < 0 || index >= list->size) {
        printf("索引超出範圍！\n");
        return -1;
    }

    Node *current = list->head;
    for (int i = 0; i < index; i++) {
        current = current->next;
    }
    return current->data;
}

// 搜尋資料是否存在
int contains(LinkedList *list, int data) {
    Node *current = list->head;
    while (current != NULL) {
        if (current->data == data) {
            return 1;  // 找到
        }
        current = current->next;
    }
    return 0;  // 沒找到
}
```

### 6️⃣ 釋放整個串列

```c
void free_list(LinkedList *list) {
    Node *current = list->head;

    while (current != NULL) {
        Node *next = current->next;  // 先記住下一個節點
        free(current);                // 釋放目前節點
        current = next;              // 移到下一個
    }

    list->head = NULL;
    list->size = 0;
}
```

> ⚠️ **注意**：釋放串列一定要用**暫存變數**記錄下一個節點的位置，否則 `free(current)` 後就找不到 `current->next` 了！

## 完整範例：整數串列管理程式

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;

typedef struct {
    Node *head;
    int size;
} LinkedList;

void init_list(LinkedList *list) {
    list->head = NULL;
    list->size = 0;
}

Node* create_node(int data) {
    Node *new_node = (Node*)malloc(sizeof(Node));
    if (new_node == NULL) {
        printf("記憶體配置失敗！\n");
        exit(1);
    }
    new_node->data = data;
    new_node->next = NULL;
    return new_node;
}

void push_front(LinkedList *list, int data) {
    Node *new_node = create_node(data);
    new_node->next = list->head;
    list->head = new_node;
    list->size++;
}

void push_back(LinkedList *list, int data) {
    Node *new_node = create_node(data);

    if (list->head == NULL) {
        list->head = new_node;
    } else {
        Node *current = list->head;
        while (current->next != NULL) {
            current = current->next;
        }
        current->next = new_node;
    }
    list->size++;
}

int pop_front(LinkedList *list) {
    if (list->head == NULL) {
        printf("串列為空！\n");
        return -1;
    }

    Node *temp = list->head;
    int data = temp->data;
    list->head = list->head->next;
    free(temp);
    list->size--;
    return data;
}

void print_list(LinkedList *list) {
    Node *current = list->head;

    printf("[大小=%d] ", list->size);
    while (current != NULL) {
        printf("%d", current->data);
        if (current->next != NULL) printf(" → ");
        current = current->next;
    }
    printf("\n");
}

void free_list(LinkedList *list) {
    Node *current = list->head;
    while (current != NULL) {
        Node *next = current->next;
        free(current);
        current = next;
    }
    list->head = NULL;
    list->size = 0;
}

int main() {
    LinkedList list;
    init_list(&list);

    int choice, value;

    do {
        printf("\n=== 鏈結串列管理 ===\n");
        printf("1. 從頭部插入\n");
        printf("2. 從尾部插入\n");
        printf("3. 從頭部刪除\n");
        printf("4. 顯示串列\n");
        printf("0. 離開\n");
        printf("請選擇：");
        scanf("%d", &choice);

        switch (choice) {
            case 1:
                printf("請輸入數值：");
                scanf("%d", &value);
                push_front(&list, value);
                print_list(&list);
                break;

            case 2:
                printf("請輸入數值：");
                scanf("%d", &value);
                push_back(&list, value);
                print_list(&list);
                break;

            case 3:
                value = pop_front(&list);
                if (value != -1) {
                    printf("刪除：%d\n", value);
                    print_list(&list);
                }
                break;

            case 4:
                print_list(&list);
                break;

            case 0:
                printf("程式結束，釋放串列...\n");
                free_list(&list);
                break;

            default:
                printf("無效選項！\n");
        }
    } while (choice != 0);

    return 0;
}
```

## 雙向鏈結串列（Doubly Linked List）

單向鏈結串列只能「從頭到尾」走訪。雙向鏈結串列增加了「指向前一個節點」的指標，可以雙向走訪。

```
NULL ⟷ [4] ⟷ [7] ⟷ [3] ⟷ NULL
        head                   tail
```

### 節點定義

```c
typedef struct DNode {
    int data;
    struct DNode *prev;  // 指向前一個節點
    struct DNode *next;  // 指向下一個節點
} DNode;

typedef struct {
    DNode *head;
    DNode *tail;  // 同時追蹤頭和尾
    int size;
} DoublyLinkedList;
```

### 雙向插入（頭部）

```c
void push_front_d(DoublyLinkedList *list, int data) {
    DNode *new_node = (DNode*)malloc(sizeof(DNode));
    new_node->data = data;
    new_node->prev = NULL;
    new_node->next = list->head;

    if (list->head != NULL) {
        list->head->prev = new_node;  // 舊 head 的 prev 指向新節點
    } else {
        list->tail = new_node;        // 如果是空串列，tail 也指向新節點
    }

    list->head = new_node;
    list->size++;
}
```

### 雙向刪除（尾部）

```c
int pop_back_d(DoublyLinkedList *list) {
    if (list->tail == NULL) return -1;

    DNode *temp = list->tail;
    int data = temp->data;

    list->tail = list->tail->prev;
    if (list->tail != NULL) {
        list->tail->next = NULL;
    } else {
        list->head = NULL;  // 刪除後串列變空
    }

    free(temp);
    list->size--;
    return data;
}
```

| 比較 | 單向鏈結串列 | 雙向鏈結串列 |
|------|------------|------------|
| 記憶體開銷 | 每個節點 1 個指標 | 每個節點 2 個指標 |
| 反向走訪 | ❌ 不行 | ✅ 可以 |
| 尾部刪除 | ❌ O(n) | ✅ O(1)（有 tail 的話） |
| 實作複雜度 | 簡單 | 較複雜（要維護兩個指標） |

## 常見面試題型

### 反轉鏈結串列

```c
void reverse_list(LinkedList *list) {
    Node *prev = NULL;
    Node *current = list->head;
    Node *next = NULL;

    while (current != NULL) {
        next = current->next;      // 先記住下一個節點
        current->next = prev;      // 反轉指標方向
        prev = current;            // prev 往前移動
        current = next;            // current 往前移動
    }

    list->head = prev;             // 更新 head
}
```

### 找出中間節點（快慢指標法）

```c
Node* find_middle(LinkedList *list) {
    if (list->head == NULL) return NULL;

    Node *slow = list->head;
    Node *fast = list->head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;          // 慢指標一次走一步
        fast = fast->next->next;    // 快指標一次走兩步
    }

    return slow;  // 當 fast 到終點時，slow 正好在中間
}
```

### 偵測環狀鏈結串列

```c
int has_cycle(LinkedList *list) {
    if (list->head == NULL) return 0;

    Node *slow = list->head;
    Node *fast = list->head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast) {
            return 1;  // 快指標追上了慢指標 → 有環！
        }
    }

    return 0;  // 快指標到終點 → 無環
}
```

## 鏈結串列的進階延伸

學會基本鏈結串列後，可以進一步學習：

| 資料結構 | 說明 |
|---------|------|
| **環狀鏈結串列** | 最後一個節點的 next 指向 head，形成環 |
| **堆疊（Stack）** | 用鏈結串列實作 push/pop 都在頭部操作 |
| **佇列（Queue）** | 用鏈結串列實作 enqueue（尾部）/ dequeue（頭部） |
| **雜湊表（Hash Table）** | 使用鏈結串列處理碰撞（chaining） |

## 本章重點整理

| 概念 | 重點 |
|------|------|
| 節點（Node） | 資料 + 指向下一個節點的指標 |
| 頭部插入 | 新節點 → head，O(1) |
| 尾部插入 | 需要走訪到最後，O(n) |
| 刪除節點 | 前一個節點跳過目標節點後 free |
| 走訪 | 從 head 開始，沿著 next 直到 NULL |
| 釋放串列 | 先用暫存變數記住 next，再 free 目前節點 |
| 雙向鏈結串列 | 多了 prev 指標，可以雙向走訪 |

## 練習題

### 練習 1：刪除特定值

撰寫函式 `void delete_value(LinkedList *list, int value)`，刪除串列中所有等於 `value` 的節點。

**提示**：需要處理連續多個相同值的情況。

### 練習 2：串列合併

給定兩個已排序的鏈結串列（遞增），將其合併為一個新的遞增串列，並回傳新串列的 head。

**提示**：類似 merge sort 的合併步驟。

### 練習 3：實作堆疊

使用鏈結串列實作一個堆疊（Stack），包含 `push()`、`pop()`、`peek()`、`is_empty()` 四個操作。

### 練習 4：約瑟夫問題（Josephus Problem）

N 個人圍成一圈，從第 1 個人開始報數，每數到 M 就將那個人淘汰，從下一個人重新報數，直到只剩一個人。使用**環狀鏈結串列**模擬這個過程。
